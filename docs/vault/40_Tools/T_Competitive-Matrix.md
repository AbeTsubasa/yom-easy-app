## 空白領域（参入余地）

1. **「ブラウザで即起動・申請不要・無料」**で **日本語ルビ＋読み上げ＋UDフォント＋行ハイライト**を統合したツールが**事実上存在しない**
2. 公的ツール（DAISY / AccessReading / UDブラウザ）はいずれも**申請制**で、日常的な「いまこのWebページが読めない」ニーズに即対応できない
3. 海外ツール（Helperbird, Immersive Reader）は**日本語ルビ自動付与に未対応**
4. 既存ふりがな拡張は**単機能**で、読み上げや色変更との統合がない
5. 商用学習アプリ（すらら等）は「教材内」読み上げで、**外部テキスト（プリント・Web）には使えない**
6. モバイル中心のアプリが多く、**PC学習環境とのシームレス連携が弱い**

## ⭐推奨ポジショニング

> **「ブラウザで即起動・登録不要・無料・日本語ルビ／読み上げ／UDフォント／行ハイライトを一括」**
> 
> DAISY等の公的支援（申請制・教科書限定）の**「すきま」を埋める日常使い**ツールとして差別化。GIGAスクール1人1台端末（Chromebook主流）と親和性が高いブラウザベース実装。
