# EM_Exam-Support-Microcopy — 受験生支援のマイクロコピー集

> 受験生（中学受験・高校受験・大学受験）を対象とした声かけ・アプリ内表示のマイクロコピー集。査読研究と塾・予備校の実務知見に基づく。
> 根拠：[[Phase1C_Psychology_Notes#2. 受験生メンタルケアの心理学]]、[[TL_M8-Exam-Mental-Care]]

## 設計原則（受験生向け）

1. **押し付け励まし禁止**：「頑張れ！」「絶対大丈夫！」は逆効果（プレッシャー強化）
2. **適度な緊張を肯定**：Yerkes-Dodson逆U字。「緊張するな」ではなく「緊張は実力発揮の証拠」
3. **方略帰属**：失敗を「能力」ではなく「方略・戦略」に帰属させる（Weiner）
4. **共感が先、提案は後**：問題解決を急がない
5. **生活基盤の安定を最優先**：睡眠・食事・体調

## アプリ内表示のマイクロコピー（時期別）

### 直前期モード（試験1週間前以内）

#### アプリ起動時の励まし（任意・ON/OFF可能）
```
お疲れさま。

今日もここを開いてくれてありがとう。
無理せず、自分のペースで。
```
※「がんばれ！」のような直接的励ましは使わない。

#### 設定変更時
```
✓ 設定を変えたね
読みやすさは人それぞれ。
本番に近い設定でも一度試しておこう。
```

### 試験前日モード（任意切替）

#### 起動時の表示
```
明日、ここまで来られた自分を、まず認めよう。

今夜は早く寝るのが一番の準備。
眠れなくても、横になって目を閉じているだけで疲れは取れる。
```

#### 4-7-8呼吸法ガイド
```
緊張してきたら、深呼吸を3セット。

4秒かけて吸って
7秒止めて
8秒かけてゆっくり吐く

[ガイド開始]  [今はいい]
```

### 試験当日朝モード

#### 起動時の表示
```
おはよう。

朝ごはん、いつも通り食べた？
深呼吸を3回。
持ち物は揃ってる？

行ってらっしゃい。応援してる。
```

#### 「自信がない」とテキスト入力された時（v2機能）
```
自信がない、って言葉にできただけで十分。
中に溜め込まないでくれてありがとう。

自信は結果が出る前に持つのは難しいもの。
だから、今日できることだけやろう。

これまで君がやってきたことは、ちゃんと積み重なってるよ。
```

### 試験後モード

#### 合格時
```
✓ 合格おめでとう

数字や合否より、
ここまであなたが続けてきたことが
何より価値ある経験。

少しゆっくりしてね。
```
※「合格したから偉い」式の評価語を避ける（プロセス褒め）

#### 不合格時
```
✓ 設定済み（応援モード）

まずは、本当にお疲れさま。

頑張ったことは、結果と関係なく事実として残る。
今日は何もしなくていい。

話したくなったら、また来てね。
```
※「次があるから」「だから言ったでしょ」絶対NG（東京すくすく・ベネッセ推奨）

## 講師向け対話スクリプト集（場面別）

### A. 試験1週間前 — 「不安で勉強が手につかない」

```
講師：不安で集中できないんだね。それ、本気で取り組んできた証拠でもあるよ。

無理に詰め込もうとせず、今日は「今までやったノートを1冊だけ見返す」のだけにしようか。
（焦点を絞る・限定）

それと、生活リズムを整えるのも勉強と同じくらい大事。
睡眠だけは確保しよう。
```

### B. 試験前日 — 「全然眠れそうにない」

```
講師：前日の緊張、当然のことだよ。

眠れなくても、横になって目を閉じているだけで体は休まる。
スマホは枕元から離して、明日のことは考えない時間を作ろう。

緊張してきたら、4-7-8呼吸を3セット試して。
```

### C. 試験当日朝（送り出し）

```
講師（保護者役）：
「いつも通りでいいよ。」
「持ち物だけ確認しよう。」
「終わったら、好きなもの食べに行こうか。」
```
※「絶対受かれ」「失敗するな」絶対NG

### D. 試験中の科目間休憩（オンライン助言できる場合）

```
講師：1科目失敗したかもしれない、と感じても、
今は次の科目だけ考えよう。

呼吸を3回深く。水を一口。
過ぎたことを採点する権利は、自分にはない。
```

### E. 結果発表後（不合格）— 第一声

```
講師：まずは、本当にお疲れさま。

（沈黙を恐れない。話し始めるまで待つ）

頑張ったね。立派だったね。
```
※「次があるから」「気持ち切り替えて」は数日後でいい

### F. 結果発表後（不合格）— 1週間後

```
講師：話したくなったら、また聞かせて。

今は休んでいい時期。
4月以降、次の選択肢を一緒に考えるのも、まだ少し休むのも、君が決めていい。
```

### G. 模試結果が悪い時

```
講師：点数より、どこで時間が足りなかったか、一緒に見てみよう。
（方略帰属）

能力の問題じゃなくて、やり方を変えるサインだね。
この時期に弱点が見えてよかった。
本番で見つかるよりずっといい。
```

### H. 「自分はバカだから受からない」

```
講師：「バカだから」って自分にレッテル貼っちゃってるんだね。
（複雑な反映）

これまでで、自分でも「あれは頑張れたな」って思える経験を、一つだけ挙げるとしたら何？
（例外探し、SFA）
```

### I. 「もう無理」と限界を訴えた時

```
講師：そう感じるくらい疲れてるんだね。
（反論しない）

今、何が一番きつい？
（評価せず状況整理）

今日は何もしなくていい。
勉強より、休息と睡眠と食事を優先しよう。
```

- 朝起きられない（**自殺SOSサインの一つ**）
- 「死にたい」「消えたい」発言
- 自傷行為の痕跡

## ディスレクシア当事者の受験生に特有の声かけ

### 合理的配慮を恥じている時
```
「メガネをかける人を特別扱いとは言わないでしょ。同じことだよ。
合理的配慮はあなたの権利。胸を張って使っていい。」
```

### 読み上げ配慮を申請するかどうか迷っている時
```
「『楽をしている』んじゃなくて、『あなたに合った形で力を発揮する』だけだよ。
読み上げで聞いて理解する力は、目で読んで理解する力と同じ価値がある。」
```

### 「漢字が書けないからもう諦めたい」
```
「漢字を書く力は、あなたの賢さの一部にすぎない。
読みのルートが違うだけで、考える力とは別の話。
今は音読み・意味で覚えていく方法もある。」
```

## エクスプレッシブ・ライティング（試験10分前用）

アプリ実装案（5分書き出しテンプレ）：

```
試験まであと10分。

今、頭の中にある「気がかり」「不安」を、
何でもいいから5分間、書き出してみよう。

正しい文章じゃなくていい。
箇条書きでも、単語だけでも、ぐちゃぐちゃでもいい。

研究で、試験直前にこれをすると
不安が和らいで実力が出やすくなることが分かっています。
（教育心理学研究 2022, 中学生で検証）

[5分タイマー開始]  [今はいい]
```

書き出した内容は**自動で消去**（プライバシー）か、ユーザー任意の保存。
