# EM_Tone-Principles — トーン規約・NG/OKワードリスト

> マイクロコピーを書く前に確認するチェックリスト。実装者・コピーライター向け。

## トーン規約（全体）

| 軸 | 採用するトーン |
|---|---|
| **人称** | 「あなた」または無人称。「君」「お子さん」は避ける |
| **敬体／常体** | **敬体（ですます）**を基本に。ただし押し付けがましくならないよう柔らかく |
| **文長** | 短文。1文20字以内が目安。長文は分解 |
| **絵文字** | 控えめに。使うなら ◯ ✓ など中立的なものに限定。😊😅 等の感情表現絵文字は使わない |
| **ビックリマーク** | **基本使わない**。使うと「がんばろう！」式に近づく |
| **疑問符** | やわらかい確認に使う。「読みやすくなったかな？」 |
| **省略形・若者言葉** | 使わない。中立的な日本語で |
| **専門用語** | 初出時に分かりやすく。「ディスレクシア」より「読み書きが苦手な特性」を優先 |

## NGワードリスト（使わない）

### 病気・治療系
- 治す／治療／矯正／改善／回復／治る／健常／障害（医療文脈）
- リハビリ／訓練／練習／鍛える／克服／立ち直る
- 症状／患者／診断（アプリ内では避ける）

### 努力・道徳系
- がんばる／努力／忍耐／辛抱／我慢／めげずに
- 乗り越える／立ち向かう／挑戦する（押し付け文脈で）
- 負けない／諦めない／強くなる

### 評価・上下系
- 正しい／正解／間違い／不正解／失敗
- できる／できない（直接的に）
- レベル／ランク／上達／成長（過度な進歩主義）

### 感情・大袈裟系
- すごい！／素晴らしい！／よくできました！
- 輝く／光る／きらめく／羽ばたく
- 才能／天才／ギフテッド（直接的に）

### 啓発・押し付け系
- 一緒にがんばろう／応援しています
- あなたならできる／信じています
- 強みを発揮しよう

## OKワードリスト（推奨）

### 「合う」「選ぶ」系
- 合う／合わない／ぴったり／しっくり
- 選ぶ／選択／決める／設定する
- 試す／変える／組み合わせる／調整する
- 探す／見つける／自分にぴったり

### 「ペース」「リズム」系
- ペース／リズム／自分のペース／ゆっくり／自分の時間
- マイペース／自分のやり方／自分のスタイル

### 「やわらかい肯定」系
- いい感じ／ちょうどいい／読みやすそう
- 設定できたね／変えてみたね／見つかったね
- これでどう？／こんな感じかな

### 「事実ベース」系
- ここまで読めたね（時間・量の事実）
- 設定を変えてみたね（行動の事実）
- 〇分集中できたね（時間の事実）

### 「強み」系（さりげなく）
- 違った見方／別の角度／独自のアイデア
- パターンを見抜く／つなげる／視覚的に考える
- （Made by Dyslexiaの6スキルから派生）

## サンプル変換

| ❌ NG | ✅ OK |
|---|---|
| もう一度がんばろう！ | 別の設定も試してみる？ |
| 正しく読めました！ | ここまで読めたね |
| 練習を続けましょう | 自分のペースでどうぞ |
| ディスレクシアの症状を改善 | あなたに合う読み方を見つける |
| エラーが発生しました | うまく行かなかったみたい。別の方法を試してみよう |
| すごい！満点です！ | 〇分で〇〇字読めたね |
| 強みを発揮しよう！ | あなたなりのやり方で |
| 治療効果のあるフォント | 多くの人が試しているフォント |

## チェックリスト（実装前に確認）

コピーを1つ書くたびに以下を確認：

- [ ] NGワードを1つも含んでいない
- [ ] ビックリマークがない（または最小限）
- [ ] 敬体だが押し付けがましくない
- [ ] 「あなた」または無人称
- [ ] 事実ベース（評価ではない）
- [ ] 短い（20字以内が目安）
- [ ] 友達に見られても恥ずかしくない（[[V_Design-Dos-Donts]]）
