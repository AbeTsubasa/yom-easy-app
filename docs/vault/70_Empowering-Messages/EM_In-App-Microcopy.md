# EM_In-App-Microcopy — アプリ内マイクロコピー集

> アプリ全画面のボタン・ラベル・ツールチップ・ヘルプテキストのマイクロコピー一覧。

## ボタン・操作系

| 場面 | ❌ NG | ✅ OK |
|---|---|---|
| 読み上げ開始 | 「練習を始める」 | **「読み上げる」** |
| 設定変更 | 「あなたを設定する」 | **「設定」** |
| ふりがな ON | 「ふりがな機能を有効にする」 | **「ふりがなを付ける」** |
| 行ハイライト ON | 「リーディング補助を有効にする」 | **「行をハイライト」** |
| カスタマイズ詳細 | 「上級者設定」 | **「もっと細かく設定」** |
| プリセット呼び出し | 「治療プリセット」 | **「おすすめの組み合わせ」** |
| リセット | 「初期化」 | **「最初の状態に戻す」** |
| 印刷 | 「印刷する」 | **「印刷する」**（自然な日本語） |
| 共有 | 「設定を医師に共有」 | **「この設定を共有」** |

## サイドメニュー・ラベル

```
📝 テキストを貼り付け
📂 ファイルを開く
🎨 見た目を変える
   ├── フォント
   ├── 文字の大きさ
   ├── 行間 / 字間
   └── 背景色 / 文字色
🔊 読み上げ
   ├── 速さ
   └── 一時停止 / 再開
✏️ ふりがな
   └── 学年で選ぶ
🪞 集中する
   ├── 行にライト
   └── 蛍光ペン
⚙️ 詳しい設定
💾 保存・印刷
```

> 「障害向け」「特別支援」を一切示さない。普通のテキストエディタ風。

## ツールチップ・ヘルプテキスト

### 字間スライダー
```
字と字の間を広げます。

広げると、文字がくっついて見えにくい時に読みやすくなることがあります。
```
> 「読みにくいあなたへ」ではなく**「読みにくい時に」**と状況を主語に。

### フォント切替
```
お好きなフォントを選んでください。

人によって「読みやすい」と感じるフォントは違います。
```
> 「あなたの障害には〇〇フォント」とラベルしない。

### 読み上げ速度
```
聞きやすい速さに調整できます。

ゆっくりにしても理解度が下がるわけではありません。
むしろ多くの人がゆっくりの方が頭に入りやすいと感じています。
```
> 「遅い＝劣る」感を払拭する。

### 行ハイライト
```
今読んでいる行を目立たせます。

行を見失いやすい時、目を疲れにくくしたい時に便利です。
（CHI 2023 研究で多くの人に効果が確認されています）
```
> 控えめにエビデンスを示し、押し付けず。

### Bionic Reading 風強調（v1.1）
```
単語の最初の文字を太字にします。

この見せ方は人によって合う合わないがあります。
試してみて、合わなければOFFにしてください。
```
> **効果を約束しない**。査読研究で効果が乏しいため。

## 設定保存時

```
✓ 設定を保存しました
```
> 「素晴らしい！」「よくできました！」は使わない。事実のみ。

## 設定リセット時

```
最初の状態に戻しますか？

これまでの設定は消えます。

(戻す)  (やめる)
```

## ステータス・ヒント

### 読み上げ中
```
🔊 読み上げ中
〇〇字 / △△字

(一時停止)  (止める)
```
> 進捗バーは出すが、「完璧」「達成」を強調しない。

### ふりがな処理中
```
ふりがなを準備中...
（数秒かかります）
```

### オフライン状態
```
オフラインです。

ふりがなと読み上げは使えますが、
新しいフォントの読み込みはできません。
```
> 「機能制限」と書かず、できることを先に。
