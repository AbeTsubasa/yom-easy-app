# EM_Trauma-Informed-Microcopy — 学習トラウマに配慮したマイクロコピー

> ディスレクシア当事者の多くは、学校での失敗体験・教師の叱責・いじめ・「努力不足」誤解の積み重ねによる**学習トラウマ**を抱えている。
> 本ノートは**SAMHSA 6原則のTIC（トラウマインフォームドケア）**に準拠した、傷を再開させないアプリ内コピーを集めたもの。
> 根拠：[[Phase1C_Psychology_Notes#4. トラウマインフォームドケアと学習トラウマ]]、[[TL_M10-Trauma-Informed-Practice]]

## SAMHSA 6原則のアプリ実装

| 原則 | 実装 |
|---|---|
| 1. Safety（安全） | ログ・データはローカル保存、外部送信なし。「私は監視されない」体験 |
| 2. Trustworthiness（信頼性） | プライバシーポリシー明示、できないことを正直に書く |
| 3. Peer Support（ピアサポート） | 当事者の体験談を任意で表示 |
| 4. Collaboration（協働） | 設定はすべてユーザーが決める。「推奨」と「強制」を明確に区別 |
| 5. Empowerment, Voice & Choice | 「お任せ設定」と「自分で選ぶ」を両提示 |
| 6. Cultural Issues | 学年・文化背景に応じたコピーを切替可能 |

## 「再受傷させない」声かけの原則

3原則：
1. **責めない**
2. **比較しない**
3. **強制しない**

### アプリでの実装

#### エラー時（[[EM_Error-Empty-States]]の補強）
```
✗ NG: 「ファイル形式が正しくありません」
✓ OK: 「このファイル、うまく読み込めなかったみたい。別の方法を試してみよう」
```

#### 機能未利用時
```
✗ NG: 「まだ読み上げ機能を使っていません！」
✓ OK: 「読み上げを試してみたい？合わなかったらすぐ戻せるよ」
```

#### 設定リセット時
```
✗ NG: 「初期設定に戻されました」
✓ OK: 「最初の状態に戻したよ。また自分に合う設定を見つけよう」
```

## 「トリガー警告」の実装

過去にトラウマ化した活動を強制しないUI：

### 音読モードを開く時
```
ヒント：これから音読練習モードを開きます。

過去に音読で辛い思いをしたことがあれば、無理に使わなくて大丈夫です。
読み上げを聞くだけのモードもあります。

[音読練習を開く]  [読み上げを聞くだけ]  [やめる]
```

### 漢字テストモード（v1.1）
```
これは漢字の練習モードです。

「テスト」と聞くとプレッシャーを感じる場合は、
「練習モード」（採点なし）にも切り替えられます。

[テストモード]  [練習モード（採点なし）]  [やめる]
```

### 時間制限がある機能
```
時間制限は外せます。

自分のペースでやりたい時は、タイマーをOFFにしてください。
[タイマーOFF]
```

## 「パス」「ヘルプ」コードの実装

オンライン授業・対話シミュレーターで、生徒がいつでも撤退できる仕組み：

```
（画面下部に常時表示）

[パス]：このタスクをスキップ
[ヘルプ]：先生にメッセージ
[休む]：5分タイマーで離席
```

## 学習トラウマを話された時のシステム応答（v2 機能）

ユーザーがフリーテキストで過去の傷つき体験を書き込んだ場合（任意機能）：

```
（自動応答）

話してくれてありがとうございます。

辛い経験を言葉にすることは、簡単ではないことです。
ここで読んだ内容は、外部には一切送信されません。

もしもう少し話したい場合、信頼できる大人に伝えることもできます：
[塾長／講師に共有する]
[スクールカウンセラーに繋ぐ]
[専門相談窓口を見る]

または、ただ書き出しただけにする：
[このまま閉じる]
```

※ AIや講師が直接的に深い心理的介入を行わない。**専門機関接続のゲートウェイ**として機能。

## 「先生はバカにしてないって本当？」レベルの疑念対応

ディスレクシア当事者は過去の経験から教員・講師全般を信用しにくい。アプリ内のFAQ等に：

```
Q: このアプリ、本当にディスレクシアの私を「治そう」としていない？

A: はい、していません。

このアプリは、文字を「あなたに合う見え方」に変えるだけのツールです。
「読み方は人それぞれ違う」のが研究で分かっていて、
あなたに合う方法を探すお手伝いをしています。

過去にどんなにひどい言葉を言われてきても、
あなたの読み方は「変えるべきもの」ではありません。
ただ、「合う形に整えるもの」です。

参考：藤堂栄子「ディスレクシアは多様性、活かす対象」（NPO EDGE）
参考：Made by Dyslexia「Dyslexic Thinking is a Skill」
```

```
あなたのことが心配です。

「死にたい」「もう疲れた」と感じる時、

今すぐ話せる相手：

　0120-279-338（24時間無料）

　0570-783-556

[電話する]  [今は閉じる]
```

## 「特別扱い」を恥じている時の声かけ

```
合理的配慮を受けることを、
「特別扱い」「ずるい」と感じる必要はありません。

メガネをかける人を「特別扱い」とは言わないでしょう。
それと同じです。

あなたが力を発揮するための、当然の権利です。
```

## 不登校から復帰する時の声かけ

```
来てくれてありがとう。

今日は10分でも、5分でも、
来てくれただけで十分。

途中で帰っていいし、横になりたかったら言って。
テストやドリルは、今日はやらない。
お話だけしよう。
```

## アプリ全体のトーン規約（TIC準拠）

[[EM_Tone-Principles]]に以下を追加：

| 場面 | 配慮 |
|---|---|
| 達成・記録 | 「○日連続！」「○問正解！」のような誇示しない（「努力強迫」誘発） |
| 通知 | プッシュ通知で「今日も勉強しよう」と煽らない |
| 統計表示 | 学習時間が短い日があっても「お疲れさま」とだけ表示、責めない |
| ストリーク（連続日数） | 機能として実装しない（連続が途切れた時の罪悪感誘発） |
| 比較 | 他ユーザーとの比較機能を実装しない |
