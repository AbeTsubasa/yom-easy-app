# EM_Psychology-Foundations — 声かけUI/UXの心理学的根拠

> 本アプリの声かけ設計（70_Empowering-Messages/）は、**Made by Dyslexia 哲学・NPO EDGE藤堂栄子氏の発信**に加え、**学術心理学の3大理論**で裏付けられている。「なぜこの声かけが効くのか」を講師・開発者が理論的に説明できるようにするための根拠ノート。

## 3大理論ダイジェスト

### 自己決定理論（SDT — Deci & Ryan）

#### 3つの基本的心理欲求
- **自律性（Autonomy）**：自分で選び、自分の意思で行動している感覚
- **有能感（Competence）**：「できる」「上達している」感覚
- **関係性（Relatedness）**：他者とつながり、認められている感覚

#### 動機づけの内在化連続体
無動機 → 外的調整 → 取り入れ的調整 → **同一化的調整（重要だと納得）** → 統合的調整 → 内発的動機づけ

#### 西村多久磨ら（2011）の重要知見
- **同一化的調整がメタ認知的方略を介して学業成績を予測**
- 純粋な内発的動機（楽しさだけ）は必ずしも成績を予測しない
- → アプリのメッセージは「楽しいよ」より「**あなたにとってこの読みやすさが大事**」を強調すべき

### マインドセット理論（Carol Dweck）

#### Mueller & Dweck（1998）— 古典研究
- 能力ほめ（"You must be smart"）→ Fixed Mindset誘発、失敗後に持続性低下
- 過程ほめ（"You must have worked hard"）→ Growth Mindset誘発、粘り強さ維持

#### ⚠ False Growth Mindset（Dweck本人の2015年警告）
Dweckが *Education Week* で自身の理論の誤用を警告：

| 誤用 | 何が問題か |
|---|---|
| 戦略なしの「努力ほめ」 | "good effort" だけは空っぽ |
| マインドセットだけ持たせ戦略は教えない | 無責任 |
| 「努力すれば誰でもできる」 | **ディスレクシアに対し致命的に有害** |

→ アプリでは「努力ほめ」ではなく**「戦略ほめ・工夫ほめ」**へ。

#### Effective Effort（効果的な努力）
- 同じ方法で頑張ることではなく、**うまくいかなければ別の方法を試す**こと
- 助けを求めること、ツールを使うこと、戦略を変えることがすべて含まれる

### 自己効力感（Self-Efficacy — Bandura, 1977）

#### 4つの源
1. **遂行行動の達成（成功体験）**：最も強力
2. **代理経験**：似た他者ができるのを見る
3. **言語的説得**：信頼できる他者からの励まし
4. **生理的・情動的状態**：緊張・疲労の解釈を変える

#### 効力期待 vs 結果期待（決定的に重要）
- **効力期待**：「自分は行動を遂行できる」
- **結果期待**：「その行動をすれば結果が得られる」
- ディスレクシア児は「努力しても成績が上がらない」経験から**結果期待が低下**
- → **効力期待だけ煽っても動機は回復しない**。結果期待を「読みやすくなったね」式に高める必要がある

## 既存の[[EM_Design-Principles|声かけ設計原則]]との対応

| 設計原則 | 学術的根拠 |
|---|---|
| 1. 「治す」NG → 「合う方法を選ぶ」OK | SDT 自律性（選択の保証） |
| 2. 「がんばろう」NG → 「ここまでできたね」OK | Dweck プロセスほめ（Mueller & Dweck 1998） |
| 3. 責めない、代替を示す | TIC「再受傷させない」原則（[[TL_M10-Trauma-Informed-Practice]]） |
| 4. 大袈裟な啓発調NG → 自然な日常言語OK | SDT 関係性（押し付けは関係性を損なう） |
| 5. 強みを「埋め込む」 | Bandura 言語的説得＋代理体験 |

## 「強みベース」哲学の学術的整合性

Made by Dyslexia / NPO EDGE の「強みベース」アプローチは、単なる励まし哲学ではなく**3大理論で説明可能**：

| 理論 | 強みベースが機能する機序 |
|---|---|
| SDT | 有能感欲求が満たされる領域を確保（読み書き以外で） |
| マインドセット | 「自分は伸びる存在だ」を得意領域で確立し苦手領域に転移 |
| 自己効力感 | 別領域の成功体験が効力期待を一般化（spill-over effect） |
| 学習性無力感の反転 | 「行動と結果が連動する」経験を失敗の少ない領域で再構築 |

## 学習性無力感への配慮

#### Seligman（1967）
- 統制不可能な失敗の反復で「何をしても無駄」を学習
- 何年も読みで失敗を重ねたディスレクシア児は典型的に陥っている
- 新しい支援ツールに「どうせ無理」と試そうとしない場合、これが原因のことが多い

#### 予防・回復
- **小さな随伴経験（やってみたらうまくいった）を積ませる**
- アプリの初回オンボーディング3問テストは、まさに**「小さな成功体験」**を提供する設計

## アンダーマイニング効果（Deci 1971）への配慮

- 内発的に動機づけられた行動に外発的報酬を与えると、動機が外発化
- 報酬がなくなると行動も消える
- → **「シール」「ご褒美」「バッジ」「スターポイント」**は安易に導入しない
- アプリではゲーミフィケーション系の派手な祝福を避ける（→ [[EM_Celebration-Moments]]）

## 神戸大学・RIETI DP 22-J-037（2022）— 日本での実証研究

「褒め方、叱り方が子どもの将来に与える影響」：

| 接し方 | 子の将来への影響 |
|---|---|
| 「頑張ったね」と過程ほめ | **自己決定度・安心感が最も高い** |
| 「褒美をもらった」 | **自己決定度が最も低い**（アンダーマイニング） |
| 「次は頑張ろうね」と励まされた | 「どうしてできないの」と叱られた人より自己決定度・安心感が高い |

→ アプリの声かけは**過程ほめ・戦略ほめ**を中核に。
