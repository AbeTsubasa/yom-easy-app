# M8: 受験生メンタルケアの心理学

## 学習目標

- 試験不安の主要理論（STAI / Worry-Emotionality / Yerkes-Dodson）を理解する
- エクスプレッシブ・ライティング等のエビデンスベース介入を実践できる
- 本番期の時期別声かけ・燃え尽き／受験うつへの対応を獲得する

## キーポイント

1. **試験不安は「Worry（認知）」と「Emotionality（情動）」の2成分**。成績低下に直結するのはWorry
2. **エクスプレッシブ・ライティング**は中学生で実証エビデンスあり（教育心理学研究 2022）
3. **不合格時は「共感→努力承認→沈黙の時間」**。問題解決を急がない
4. **燃え尽き・受験うつの自殺リスクサイン**を見逃さない（特に**朝起きられない**）
5. **「がんばれ！」式の押し付け励まし**は逆効果

## 1. 試験不安の理論

### 1.1 Spielberger STAI（状態・特性不安）

| 種別 | 内容 |
|---|---|
| 状態不安 | 特定場面で一過性（試験当日のドキドキ） |
| 特性不安 | 性格傾向としての慢性的な不安傾向 |

日本では水口公信・下仲順子・中里克治により標準化、2021年に再標準化版（Form-JYZ）発売。

### 1.2 Worry vs Emotionality（Liebert & Morris 1967, Sarason）

| 成分 | 内容 | 成績への影響 |
|---|---|---|
| **Worry（認知）** | 失敗への懸念、結果への思考的占有 | **直接的に成績低下** |
| **Emotionality（情動）** | 心拍上昇、発汗、吐き気 | 間接的 |

→ **「緊張するな」と言うより、「不安な思考を書き出して整理する」介入の方が効果的**。

### 1.3 Yerkes-Dodson 法則（逆U字）

- 覚醒水準とパフォーマンスは逆U字
- **適度な緊張は実力発揮に必要**
- 過度な緊張は成績低下、しかし完全リラックスも成績低下

#### 講師への含意
- 「緊張するな」「リラックスして」は誤指導
- 正しい：**「緊張するのは本気で取り組んでいる証拠」「適度な緊張があるから力が出る」**

### 1.4 「頭が真っ白」のメカニズム

ワーキングメモリ説（Eysenck 注意制御理論）：
- 不安思考がWMの容量を占有
- 課題に使える認知資源が減少

→ 不安を**外在化（書き出す）**ことでWM容量を解放するのがエクスプレッシブ・ライティングの理論的根拠。

## 2. 和文の主要研究

| 文献 | 知見 |
|---|---|
| **児玉裕巳・石隈利紀（2015）**『教育心理学研究』63(3) | 中高生1,361名で「学習不安」因子を抽出 |
| **⭐『教育心理学研究』70(3), 2022** | **中学生のテスト不安に対する短期構造化筆記（エクスプレッシブ・ライティング）の効果**：不安緩和＋数学成績向上、特に高不安群で顕著 |
| **元永拓郎**「大学受験予備校における自殺予防活動」 | 駿台等での実践、予備校全体をコミュニティ・メンタルヘルス・サービスとして枠組み化 |
| **『教育心理学研究』33(3), 1985** | Weiner原因帰属理論の国内検証 |

## 3. 試験不安への介入（エビデンスベース）

### 3.1 認知行動療法（CBT）
- 国立精神・神経医療研究センター（NCNP）認知行動療法センター（2011年設立）が中核
- 子どもの不安症ではCBTは薬物療法と同等以上のエビデンス
- 不安症・うつへの「統一プロトコル（UP）」

### 3.2 ABC理論（Albert Ellis）
- **A**ctivating event → **B**elief → **C**onsequence
- 不合理な信念を論駁し、合理的信念に置換

#### 受験での典型イラショナル・ビリーフ
- 「絶対に落ちてはいけない」
- 「失敗したら人生が終わる」
- 「皆に勝たねば価値がない」

#### 論駁例
- 「『絶対に』『〜ねば』は本当か？落ちた人の人生はそこで終わっていない事実は？」

### 3.3 リフレーミング
- 「緊張している」→「**本気で取り組んでいる証拠**」
- 「不合格」→「**この時期に弱点が見えた、本番で見つかるよりよかった**」
- **無理にポジティブ化せず、視点を増やす**のがコツ

### 3.4 マインドフルネス
- スタンフォード大 マーフィ重松教授が都内予備校で実践
- 米ノースカロライナ大研究で大学生のテスト不安が有意減少
- 日本マインドフルネス学会が査読誌『マインドフルネス研究』刊行

### 3.5 呼吸法・リラクセーション
- **4-7-8呼吸法**（4秒吸う・7秒止める・8秒吐く）
- **スクエア呼吸法**（4-4-4-4）
- **腹式呼吸**
- **漸進的筋弛緩法**（Jacobson）

### 3.6 ⭐エクスプレッシブ・ライティング（筆記開示）
- Ramirez & Beilock 2011：試験直前10分書き出すと成績向上
- 日本でも中学生で実証（『教育心理学研究』2022）
- **アプリ実装の有力候補**

#### 実装プロトコル
```
試験10分前に：
- ノート or アプリで5分
- 今頭にある不安・気がかりを何でも書き出す
- 文章にならなくてOK、箇条書きでOK
- 書いたものは見直さなくてOK
```

## 4. 本番期の声かけ（時期別）

### 4.1 直前期（1週間前以内）

#### 河合塾／駿台／東進／四谷学院・湘南ゼミナール共通推奨

| ❌ NG | ✅ OK |
|---|---|
| もっと勉強しろ | 生活リズム最優先で |
| このままじゃ受からない | ここまでやってきたことを信じよう |
| ○○ちゃんは受かったのに | 他と比較しない |
| 大学に落ちたら人生終わり | 様々な道がある |

→ **「黒衣（くろこ）的サポート」**：食事・睡眠・室内環境を整える。

### 4.2 試験前日・当日朝

#### 推奨フレーズ（複数情報源で共通）
- 「ここまでやったから大丈夫」
- 「いつも通りやればいい」
- 「あなたを信じている」
- 「終わったら、好きなもの食べに行こう」（事後の楽しみを予告）

#### 避けるべき
- 「絶対受かれ」（プレッシャー）
- 「失敗するな」（恐怖喚起）

### 4.3 試験中・休憩時
- 1科目失敗を引きずる場合：**頭をリセットする呼吸法**を共有しておく
- 「次の科目だけに集中」を**事前に合言葉として擦り込む**

### 4.4 結果発表後

#### 合格時
- 努力を承認。「合格したから偉い」ではなく**「努力できたあなたが素晴らしい」**（プロセス褒め）
- 燃え尽き予防：合格後の楽しみと4月までの過ごし方を一緒に考える

#### 不合格時 ★最重要
- 第一声：**「まずは、本当にお疲れさま。」**（評価せず労う）
- 沈黙を恐れない。「話したくなったら、いつでも聞くよ」
- **「頑張ったね。立派だったね。」**（東京すくすく・ベネッセ推奨）
- 親が先回りして次の目標を励ますのは**逆効果**（ベネッセ調査）
- 親が一歩引いて、いつも通り、ほんの少し優しく接する
- 4月以降も塞ぎ込み・不眠・食欲不振が続けば、小児科・スクールカウンセラーへ

#### ❌ 絶対NGフレーズ（不合格時）
- 「だから言ったでしょ」
- 「○○ちゃんは受かったのに」
- 「次があるから」（共感前の問題解決提案）
- 「気持ち切り替えて」

## 5. 失敗体験・挫折の処理

### 5.1 Weiner の原因帰属理論

3次元：内的／外的 × 安定／不安定 × 統制可能／不可能

| 帰属 | 例 | 影響 |
|---|---|---|
| 内的・不安定・統制可能 | **「努力不足」「方略のミス」** | **次に頑張れる（推奨）** |
| 内的・安定・統制不可 | **「能力不足」** | **学習性無力感へ（危険）** |
| 外的・不安定 | 「運が悪かった」 | 動機維持に有効な場合あり |

→ 指導示唆：失敗を語らせるとき、**能力に帰属させず、方略・努力量に再帰属させる**声かけが有効。

### 5.2 学習性無力感（Seligman）
- 統制不可能な失敗の反復で「何をしても無駄」を学習
- ディスレクシア児は典型的に陥っている
- 予防：**小さな随伴経験（やってみたらうまくいった）を積ませる**

## 6. ディスレクシア・LD受験生特有の配慮

### 6.1 合理的配慮の心理的負荷
- 「特別扱いを受けている」と本人が感じる場面で、**周囲の視線がストレッサー**
- 紙資料を封筒で渡す配慮を求めた学生の例
- 教員側にも提供負担感あり（『LD研究』33巻1号）

### 6.2 セルフアドボカシー
- もじこ塾（成田あゆみ氏）：**高1・高2から早めに合理的配慮申請**
- キズキ共育塾：LD/SLD当事者は、診断書・配慮内容・自分の特性を**自分の言葉で説明できる**よう練習する

### 6.3 二次障害予防
- 国立成育医療研究センター：早期支援で二次障害予防可能
- **「読めない自分はダメ」ではなく「読みのルートが違うだけ」**のリフレーミングを早期に

## 8. メタ認知・自己調整学習（Zimmerman）

### 3要素
- **動機づけ**（やる気）
- **学習方略**（学び方）
- **メタ認知**（客観視）

### 3フェーズ
1. **予見**（目標設定・計画）
2. **遂行**（モニタリング）
3. **自己省察**（振り返り）

### 受験への応用
- 模試後の振り返り：**「能力」ではなく「方略」を点検**させる
- ジャーナリング・週次振り返りで自己効力感を維持

## 9. 保護者支援

### 9.1 過干渉問題
- 明治調査（受験生ママ）：「過干渉にならない」が気をつけることの1位（52.4%）
- ベネッセ：過干渉は主体性・自立心・自己決定感を奪う
- 東大社研×ベネッセ：**自律性支援は子の語彙力を高め、統制・直接指導は発達を抑制**

### 9.2 中学受験の親子ストレス（DeltaX調査）
- 親の約7割がストレス
- 最大要因「成績・合否への不安」（43%）
- 子の50.5%が「親をがっかりさせたくない」を動機にしてしまう

### 9.3 親への助言の核
- **「親が先回りして気分転換させようとする」のは回復力を削ぐ**
- **共感→努力承認→沈黙の時間**の順
- 不合格時の話を試験前にしている家庭は約3割（ベネッセ）

## 10. 燃え尽き・受験うつへの対応

### 10.1 リスクサイン（吉田たかよし氏、品川メンタル等）

- 朝起きられない（**「朝寝坊は自殺SOSサイン」**）
- 長文読解が急にできない
- ケアレスミスの急増
- 家族との目線回避
- 暴言・暴力
- 夜更かしスマホ
- 急な食欲変動
- 頭痛・腹痛の持続

### 10.2 自殺リスク3兆候（吉田氏）
1. 弱音の言葉のトーン変化
2. 家族との目線回避
3. 学校・塾環境の変化

### 10.3 統計
- 徳志会調査：受験生の約7割が受験うつ経験、うち**約3割が希死念慮**

### 10.4 専門機関接続（緊急時）
- 24時間子供SOSダイヤル：**0120-0-78310**
- チャイルドライン：0120-99-7777
- よりそいホットライン：0120-279-338
- NCNP 認知行動療法センター
- 国立成育医療研究センター（小児）

### 10.5 燃え尽き（合格後）
- 中学受験後の入学後発症が多い
- 「合格」という単一目標喪失で意欲低下
- 対処：英単語1日1個・10分読書など**達成しやすい小目標**で随伴経験再構築

## 11. ⭐ 場面別声かけスクリプト集

### A. 試験1週間前 — 「不安で勉強が手につかない」

```
不安があるのは、本気で取り組んできた証拠だね。
今ある力を出せれば十分だよ。

あと1週間、新しいことを増やすより、
今までやったノートを1冊だけ見返そう。
```

### B. 試験前日

```
今夜は早く寝るのが一番の勉強だよ。
眠れなくても、横になって目を閉じているだけで疲れは取れる。

明日、いつも通りでいい。
深呼吸を覚えてるね？
```

### C. 試験当日朝

```
おはよう。いつも通りでいいよ。
行ってらっしゃい。応援してる。

1科目失敗しても次に切り替えれば取り返せる。
終わったら、好きなもの食べに行こう。
```

### D. 試験中失敗時（休憩中）

```
終わった科目のことは、もう採点する権利は自分にはない。
次の科目だけ考えよう。

4-7-8呼吸を3セット → 水を一口 → 次に向かう。
```

### E. 試験後（合格）

```
合格おめでとう。
この1年あなたが続けてきたことを、見ていて誇らしかった。
（プロセス承認）

結果より、ここまで諦めなかった事実が一番大きいね。

少しゆっくりしてね。
合格後の楽しみと、4月までの過ごし方を一緒に考えよう。
（燃え尽き予防）
```

### F. 試験後（不合格）

```
（第一声）
まずは、本当にお疲れさま。

（沈黙を恐れない。話したくなったら、いつでも聞くよ）

頑張ったね。立派だったね。

（数日後）
次の選択肢を一緒に考える？
それともまだ少し休む？
（子に選択権）
```

### G. 模試結果が悪い時

```
点数より、どこで時間が足りなかったか、一緒に見てみよう。
（方略帰属）

能力の問題じゃなくて、やり方を変えるサインだね。
この時期に弱点が見えてよかった。
本番で見つかるよりずっといい。
```

### H. 「自信がない」と言われた時

```
自信がないって言葉にできたの、すごく大事だよ。
中に溜め込まないで。

自信って、結果が出る前に持つのは難しいもの。
だから、今日やれることだけやろう。

（過去の小成功を具体的に挙げる）
○月の模試で○○の単元が伸びたじゃない。あれ、あなたの実力だよ。
```

### I. 「もう無理」と言われた時

```
（反論しない）
そう感じるくらい疲れてるんだね。

（評価せず状況整理）
今、何が一番きつい？

（即対応）
今日は何もしなくていい。
勉強より休息・睡眠・食事を優先しよう。
```

→ リスクサインがあれば即時専門機関接続。

### J. ディスレクシア当事者向けに頻出

- **「読めない自分はダメ」**：「読み方のルートが多数派と違うだけ。あなたの考える力とは別の話だよ」
- **配慮申請を恥じたとき**：「メガネをかける人を特別扱いとは言わないでしょ。同じことだよ」
- **漢字書けないと泣いたとき**：「漢字を書く力は、あなたの賢さの一部にすぎない。今は音読み・意味で覚えていこう」

1. **「がんばれ！」を1週間言わずに済ますトライアル**
