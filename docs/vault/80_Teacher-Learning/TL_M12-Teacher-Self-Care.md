# M12: 講師のセルフケア — 共感疲労・代理受傷・燃え尽き予防

## 学習目標

- 共感疲労・代理受傷・バーンアウトを区別できる
- セルフケアの具体的方法を実践できる
- 「一人で抱え込まない」体制の重要性を理解する

## キーポイント

1. **共感疲労（Compassion Fatigue）**：支援者の約1/4が経験
2. **代理受傷（Vicarious Trauma）**：被害者と同じような心身反応が起こる
3. **バーンアウト**：情緒的消耗・脱人格化・個人的達成感低下の3要素
4. **「一人で抱え込まない」**：複数のサポート体制が必須
5. **講師自身が支援を受ける**ことは弱さではない

## 1. なぜ講師のセルフケアが重要か

学習トラウマを抱える生徒の支援は、講師にとって心理的負荷が大きい：

- 重い体験を聞くこと自体のストレス
- 「自分が役に立てない」感覚（無力感）
- 「もっとできたはず」という自責
- 守秘義務と専門機関接続のバランス
- 緊急時対応の継続的緊張

→ **講師自身のメンタルヘルスが、生徒支援の質を決める**。
→ セルフケアは「弱さ」ではなく「**プロフェッショナルとしての必須スキル**」。

## 2. Compassion Fatigue（共感疲労）

### 2.1 定義
- トラウマを抱える人への支援に従事する人の**約4分の1**が経験
- 看護師・介護職・カウンセラー・教員に多い
- 「共感する力」自体が消耗していく現象

### 2.2 サイン
- 慢性的な疲労感
- 感情の麻痺・関心の低下
- 仕事への意欲低下
- 不眠、頭痛、消化器症状
- 「もう聞きたくない」と感じる
- 生徒の話に共感できなくなる

### 2.3 関連概念

| 概念 | 内容 |
|---|---|
| Empathy Fatigue | 共感する力の枯渇 |
| Secondary Traumatic Stress | 他者のトラウマ体験を聞くことで生じるストレス |
| Burnout（バーンアウト） | 長期的な仕事ストレスによる消耗 |

## 3. Vicarious Trauma（代理受傷／二次受傷）

### 3.1 定義
被害者の話を熱心に聴くうちに、被害者と同じような心身反応が起こる現象。

### 3.2 症状
- フラッシュバック様症状
- 睡眠障害
- 回避行動
- 過覚醒
- 世界観の変化（「世界は危険な場所だ」）
- 信頼の喪失

### 3.3 出典
- 法政大学・廣川進の警察庁向け教材が標準的解説資料
- 国立精神・神経医療研究センター（NCNP）が「代理トラウマ予防、認識、対処法」を公開

## 4. バーンアウト（Burnout）

### 4.1 3要素（Maslach Burnout Inventory）

| 要素 | 内容 |
|---|---|
| **情緒的消耗（Emotional Exhaustion）** | 心が疲れ切ってしまう |
| **脱人格化（Depersonalization）** | 生徒を一個人ではなく「対象」として見る |
| **個人的達成感の低下（Reduced Personal Accomplishment）** | 「自分は役に立っていない」 |

### 4.2 講師・教員に特有の要因
- 業務時間外も生徒のことを考えてしまう
- 成果が見えにくい（読みの改善は時間がかかる）
- 保護者対応の負担
- 個別ニーズへの対応で標準化が難しい
- 「教育者は聖職」プレッシャー

## 5. セルフケアの具体的方法

### 5.1 ⭐ バウンダリー（境界線）の保持

- **業務時間外の連絡を制限**：保護者からのLINEは○時まで、と明示
- **メールチェックの時間帯を決める**
- **休日は完全に切る日**を作る
- **個別ケースを家で考えない時間**を作る
- 「申し訳ない」と思う必要はない。バウンダリーはプロフェッショナルの基本

### 5.2 一人で抱え込まない

- **複数の同僚・スーパーバイザーで共有**
- ケースカンファレンスを定期開催
- **守秘義務と専門機関連携のバランス**を組織で話し合う
- 一人で対応している場合は、外部のスーパーバイズを受ける

### 5.3 マインドフルネス・身体的セルフケア

- 1日10分の瞑想
- 呼吸法（4-7-8呼吸法等、生徒に教えるのと同じ）
- 適度な運動
- 十分な睡眠
- 栄養バランス

### 5.4 自身も支援を受ける

- カウンセリング（公認心理師、臨床心理士）
- ピアグループ（同業者の集まり）
- スーパービジョン
- → **「支援される側」になる経験**が、支援する力を支える

### 5.5 「自分の限界」を認める

- 「完璧な講師」を目指さない
- 自分が苦手なケースは正直に伝え、専門機関に繋ぐ
- **「申し訳ない」と感じる必要はない**

## 6. ⭐ セルフチェックリスト

定期的（月1回など）に自身をチェック：

### 身体面
- [ ] 慢性的な疲労がある
- [ ] 不眠（眠れない／眠りすぎる）
- [ ] 頭痛・肩こり・胃腸の不調が続く
- [ ] 食欲の極端な変化

### 感情面
- [ ] 生徒の話に共感しにくくなった
- [ ] イライラ、怒りっぽくなった
- [ ] 「もう聞きたくない」と感じる
- [ ] 達成感が感じられない
- [ ] 「自分は役に立っていない」と思う

### 行動面
- [ ] 仕事に行きたくない日が増えた
- [ ] 生徒や保護者への対応が機械的になった
- [ ] アルコール・カフェイン摂取が増えた
- [ ] 同僚との会話が減った

**3つ以上当てはまる場合**：燃え尽きのサインかもしれません。
- 上司・同僚に相談
- 産業医・スクールカウンセラーへ
- 外部のカウンセリング機関

## 7. 燃え尽き予防のチームアプローチ

### 7.1 組織として
- **ケースカンファレンスの定期開催**
- 1人で抱え込ませない当番制
- 緊急時対応プロトコルの整備
- 講師研修にセルフケアを組み込む

### 7.2 個人として
- 同期・同業者とのネットワーク
- 月1回のスーパービジョン
- 「相談できる人」のリスト

### 7.3 専門機関の活用
- 自治体の労働相談窓口
- 産業医制度（一定規模の事業者）
- EAP（Employee Assistance Program）

## 8. ピアサポートの活用

### 8.1 教員・支援者向けピアグループ
- 同業者の集まりで体験を共有
- 「自分だけじゃない」確認
- 解決策の交換

### 8.2 ディスレクシア支援に特化したコミュニティ
- 発達性ディスレクシア研究会
- NPO EDGE のLSA養成講座修了者ネットワーク
- atacLab ATACカンファレンス

## 9. 講師自身もケアを受ける選択肢

### 9.1 カウンセリング
- 公認心理師、臨床心理士
- 医療機関（心療内科）
- 自治体の心の相談窓口

### 9.2 ストレスチェック制度
- 50人以上の事業所では実施義務（労働安全衛生法）
- 高ストレス者は産業医面談

### 9.3 専門相談窓口（参考）
- よりそいホットライン：0120-279-338
- 日本産業カウンセラー協会
- 各自治体の労働相談センター

## 10. ⭐ 自己声かけ — 講師自身に向けて

過酷な日々の中で、自分にどう声をかけるか：

### NG（自責）
- 「私が無能だから」
- 「もっと頑張らなければ」
- 「他の先生はできているのに」
- 「申し訳ない」（過剰に）

### OK（自己肯定的）
- 「今日もここまで来た自分を認めよう」
- 「すべてに完璧に対応はできない。それでいい」
- 「他の先生にも限界がある。私だけじゃない」
- 「専門機関に繋ぐのは、私の役割の一部」
- 「セルフケアは仕事の一部」

## 11. 「治療者ではない」立場の再確認

[[TL_M10-Trauma-Informed-Practice]] の繰り返しになりますが、重要：

> **講師個人がカウンセラー役を担うのではなく、専門機関と連携する**

- 講師は「治療者」ではなく**「安全な学習空間の提供者」**
- 心理的に深い領域は専門家へ
- 「想起作業」を講師主導で行わない
- 自殺リスク等は組織として速やかに専門機関へ繋ぐ

→ **この境界線を守ることが、自分自身を守ることにもつながる**。

## 明日からできる3つのこと

1. **業務時間外の保護者連絡対応をルール化する（〇時以降は翌日対応）**
2. **月1回、同僚と「最近きついケース」を共有する場を作る**
3. **自分が「相談できる人」のリストを3名作る**

## チェックリスト

- [ ] 共感疲労・代理受傷・バーンアウトを区別できる
- [ ] 自分のバウンダリー（境界線）を明確にしている
- [ ] 一人で抱え込まない体制を持っている
- [ ] 月1回はセルフチェックを実施している
- [ ] 自分自身も支援を受ける選択肢を持っている
- [ ] 「治療者ではない」立場を理解している
