# M11: 対話技法 — MI / Rogers / Gordon / GROW / SFA

## 学習目標

- 動機づけ面接（MI）のOARSとChange Talkを使える
- Rogersの3条件と反映的聞き方を実践できる
- I-メッセージ、GROWモデル、SFAを教育場面で応用できる
- Wait-Timeを意識した間（ま）の使い方を獲得する

## キーポイント

1. **動機づけ面接（MI）**：「指示」ではなく「協働」、Change Talkを引き出す
2. **Rogersの3条件**：自己一致・無条件の肯定的関心・共感的理解
3. **I-メッセージ**：「行動＋影響＋感情」の3要素構文
4. **GROW**：Goal/Reality/Options/Will
5. **SFA**：例外探し・ミラクル・スケーリング
6. **Wait-Time**：3秒以上の沈黙で生徒の応答が劇的に改善

## 1. 動機づけ面接（Motivational Interviewing, MI）

### 1.1 理論基礎（Miller & Rollnick）
- 依存症治療の臨床研究から開発された「変化への動機を引き出す協働的会話スタイル」
- 原典は1991年初版、現在第3版（邦訳：原井宏明・岡嶋美代訳、星和書店2019）
- **「指示」ではなく「協働」**を中核に据え、両価性（変わりたい／変わりたくない）を扱う

### 1.2 4つのスピリット（PACE）
- **P**artnership（協働）
- **A**cceptance（受容）
- **C**ompassion（思いやり）
- **E**vocation（喚起）

### 1.3 4つのプロセス
1. **Engaging（かかわる）**：信頼・援助関係の確立
2. **Focusing（フォーカスする）**：取り組む目標を協働で明確化
3. **Evoking（引き出す）**：クライエント自身のチェンジトークを引き出す
4. **Planning（計画する）**：本人の準備が整って初めて具体策へ

### 1.4 ⭐ OARS の具体例（教育場面）

| 文字 | 技法 | 教育場面の例 |
|---|---|---|
| **O** | **Open question**（開かれた質問） | 「最近、勉強で一番モヤモヤしてるのはどんなところ？」 |
| **A** | **Affirmation**（是認） | 「テスト前、自分で計画立てて取り組んだのは大きいね」※評価ではなく事実承認 |
| **R** | **Reflection**（聞き返し） | 単純な反映と複雑な反映 |
| **S** | **Summary**（要約） | 複数発言をつないで要点を返す |

#### 要約の原則
- **「チェンジトークで終える」**
- **「『でも』ではなく『一方で／そして』でつなぐ」**

### 1.5 Change Talk vs Sustain Talk

#### 準備のChange Talk（DARN）
- **D**esire：「もし英語ができるようになったら、何が変わると思う？」
- **A**bility：「これまで、できるようになった経験で似たものはある？」
- **R**easons：「読み上げ機能を使う理由は？」
- **N**eed：「今、何が必要だと感じる？」

#### 行動のChange Talk（CAT）
- **C**ommitment：「やってみる」「決めた」
- **A**ctivation：「来週まずやってみてもいいかなと思うのはどれ？」
- **T**aking steps：「もうやってみた」

#### Sustain Talk（維持トーク）への対応
「どうせ無理」「面倒くさい」に**正論で反論せず、反映で受け止める**：
- 「面倒くさいよね（反映）。じゃあ、面倒くさいけれど、それでもやろうかと思うとしたら、どこから始める？」

### 1.6 日本での実践
- **北田雅子**（札幌学院大）：『医療スタッフのための動機づけ面接 逆引きMI学習帳』医歯薬出版、AndMIlab主宰
- **日本動機づけ面接学会**：年次大会・トレーニング
- **MITI 3.0日本語版**予備的検討（『行動療法研究』）

## 2. 能動的傾聴（Carl Rogers）

### 2.1 3条件（必要十分条件）

| 条件 | 内容 |
|---|---|
| **自己一致（Congruence／純粋性）** | 援助者が自分の感情と表現を一致させる |
| **無条件の肯定的関心（Unconditional Positive Regard）** | 判断せず、その人を一人の人間として尊重する |
| **共感的理解（Empathic Understanding）** | 相手の世界を「あたかも自分のことのように」感じ取る |

### 2.2 反映的聞き方の実例

#### 単純な反映（Simple Reflection）
オウム返し：
- 「英語、もう無理。」→「もう無理、って感じてるんだね」

#### 複雑な反映（Complex Reflection）
言い換え・感情の名付け：
- 「英語、もう無理。」→「単語が頭に入らなくて、自信なくしてる感じかな」

#### 反映のコツ
反映は「問い返し」より**「言い切り型（語尾を上げない）」**のほうが、相手は探索を続けやすい（MIの原則）。

### 2.3 日本での応用
- **日本産業カウンセラー協会**：「話し手・聴き手・観察者」の体験方式
- 厚労省こころの耳「積極的傾聴法」解説

### 2.4 よくある失敗パターン

| 失敗 | 内容 |
|---|---|
| 評価的応答 | 「えらいね」「すごいね」 |
| 早すぎる助言 | 「こうした方がいい」 |
| 即同意 | 「分かります」（阿川佐和子も禁句として指摘） |
| 自分の経験談に流す | 「先生も昔ね…」 |
| 質問攻め | 立て続けに尋ねる |

## 3. I-メッセージ（Gordonモデル）

### 3.1 トマス・ゴードンと親業/教師学
- ゴードン（シカゴ大博士／臨床心理学者）が1962年に開始した P.E.T.（Parent Effectiveness Training）／T.E.T.（Teacher Effectiveness Training）
- 日本では**親業訓練協会**が普及
- 3本柱：①能動的な聞き方 ②I-メッセージ ③勝負なし法（対立解消）

### 3.2 構文：3要素

**①行動**（非難なき事実）＋**②影響**（自分への具体的影響）＋**③感情**（私はこう感じる）

### 3.3 教育場面実例

#### ❌ You-メッセージ
「またサボったな！やる気あるのか？」

#### ✅ I-メッセージ
「宿題が出されていないのを見ると（行動）、次の授業の組み立て方が分からなくなって（影響）、私はちょっと困ってるんだ（感情）」

#### 別例
- ❌「またうるさくして！」
- ✅「みんなが集中して書いているときに大きな声がすると（行動）、僕も他の生徒も集中が切れてしまって（影響）、もったいないなと思うんだ（感情）」

## 4. コーチング — GROWモデル（ジョン・ホイットモア）

| 文字 | 質問例 |
|---|---|
| **G**oal（目標） | 「最終的にどうなったら『成功』と言える？」 |
| **R**eality（現状） | 「いま、何ができていて、何が壁になってる？」 |
| **O**ptions（選択肢） | 「やり方は他に何が考えられそう？」 |
| **W**ill（意思・行動） | 「では、明日から最初の一歩は何にする？」 |

### 4.1 日本での教育コーチング
- **一般社団法人 日本青少年育成協会・教育コーチング**：「傾聴・質問・承認」の3技法
- **NPO法人日本スクールコーチ協会**：学校現場向け展開
- **駿台**：2024年「**大学受験コーチング検定**」開始（4段階／30回テスト＋修了試験）、高校教員・予備校講師向け

## 5. 解決志向アプローチ（SFA / SFBT — de Shazer）

### 5.1 主要技法

#### 例外探し（Exception Finding）
- 「問題が少しでもマシだった日は？そのときは何が違った？」
- **失敗ではなく成功の小さなきっかけ**を発見

#### ミラクル・クエスチョン
- 「今夜眠っている間に奇跡が起きて問題が全部解決したら、翌朝、最初に何で気づく？」
- 解決後の状態を具体化

#### スケーリング・クエスチョン
- 「最悪が0、理想が10だとして、今はいくつ？1つ上げるには何が必要？」
- 状態を数値化し、現実的な変化を可視化

#### コーピング・クエスチョン
- 「それだけ辛いのに、どうやって今日ここまで来られた？」
- すでに持っている対処資源を意識化

### 5.2 スクールカウンセリング応用
- **黒沢幸子ほか**『ブリーフ学校カウンセリング』ナカニシヤ出版
- **森俊夫ほか**『学校で役立つブリーフセラピー』金剛出版
- 文科省『スクールカウンセラー等活用事業 実践活動事例集』

## 6. 非言語コミュニケーション

### 6.1 メラビアンの法則の正しい理解

- メラビアン（Albert Mehrabian, 1971）の実験は**「言語・聴覚・視覚の3情報が矛盾している場合、受け手は非言語に重みを置く（言語7%／聴覚38%／視覚55%）」**という**限定条件下の結果**
- **誤用**：「人の印象は見た目が9割」とコミュニケーション全般に拡張する解釈は誤り
- 正しい示唆：内容と表情・声が矛盾していると不信感を生む。情報伝達自体は言語が主役

### 6.2 教育場面での実装
- 視線、姿勢、声のトーン、距離
- 「忙しそうな姿勢で『どうした？』と聞く」と矛盾メッセージになる
- **生徒と話す時は手元の作業を止める**

## 7. アサーション（DESC法）

### 7.1 構文：4要素
- **D**escribe（描写）：「今日提出予定だった課題、まだ届いていないんだ」（事実のみ）
- **E**xplain/Express（説明・表現）：「私はこのままだと授業計画を変えなきゃで、ちょっと焦ってる」
- **S**pecify（提案）：「明日の朝までにLINEで写真送ってくれるとすごく助かる」
- **C**hoose（選択）：「もし難しそうなら、いつなら出せそうか教えて」

### 7.2 日本での第一人者
- **平木典子**：『アサーション・トレーニング』『アサーション入門』講談社現代新書

## 8. 質問の技術

### 8.1 オープン質問 vs クローズド質問
- **オープン**で自己探索を促す
- **クローズド**は確認・限定に使う

### 8.2 5W1H の使い分け
- **「Why（なぜ）」は非難に響きやすい** → 多用しない
- 言い換え：「**どんなとき**」「**何が**」「**どうしたら**」

### 8.3 視点を変える質問（リフレーミング系）
- 「同じ立場の友達がそう言ってたら、何て声をかける？」
- 「半年後の自分なら、いまの自分に何を勧める？」

### 8.4 焦点を絞る質問
- 「具体的に、どの単元？」
- 「どの瞬間が一番きつかった？」

## 9. ⭐ Wait-Time研究（Rowe 1972, 1986）

### 9.1 知見

教師が質問後の沈黙を**1秒→3秒以上**にすると、以下が増加：
- 生徒の応答長
- 自発的応答数
- 自信
- **「考えの遅い子」からの応答**

### 9.2 Wait-Time 1 と Wait-Time 2
- **Wait-Time 1**：質問後〜回答開始まで
- **Wait-Time 2**：回答後の追加の間（次の発言までの間）

両方を意図的に確保する。

### 9.3 ディスレクシア生徒への含意
- 読み上げる時間、考える時間が長い
- **Wait-Time 1・2 をデフォルト5秒設定**で意識づけ

### 9.4 カウンセリングにおける沈黙
- 自己洞察中・言葉を選んでいる時の沈黙は**「待つ」べきもの**
- 拒否・抵抗の沈黙は問いを変える

## 10. 日本特有の対話文化への配慮

### 10.1 高文脈文化
- 日本人のコミュニケーションは「察する」「行間を読む」「本音と建前」を含む
- 生徒の沈黙＝拒否とは限らず**「言葉にできない」「察してほしい」**状態の場合がある

### 10.2 実装上の配慮
1. **「言わせる」より「言ってもいい場をつくる」**
2. 反映は控えめに、断定形は避ける（語尾「〜なのかも」）
3. 要約のあと**「ここまでで違うところある？」と確認質問**を必ず入れる

## 11. ⭐ 場面別対話スクリプト集（統合）

### A. 「やる気が出ない」と言われたとき

| ❌ NG | ✅ OK（反映＋スケーリング） |
|---|---|
| 「やればできるよ」 | 「やる気が出ない、しんどい感じなんだね（反映）。0が全然出ない、10が満タンだとしたら、今はどれくらい？……2か。じゃあ、これが3になるとしたら、何が小さく変わってると気づける？」 |

### B. 「自分はバカだから」と言われたとき

| ❌ NG | ✅ OK（複雑な反映＋例外探し） |
|---|---|
| 「そんなことないよ」（直接否定は両価性を強める） | 「『バカだから』って自分にレッテル貼っちゃってるんだね（反映）。これまでで、自分でも『あれは頑張れたな』とちょっと思える経験を一つだけ挙げるとしたら？」 |

ディスレクシア文脈の配慮：「読み書きの仕組みが他の人とちょっと違うだけで、知能とは別だよ」と事実を短く伝えてから探索に戻す。

### C. 「もう辞めたい」と言われたとき

| ❌ NG | ✅ OK（受容＋両価性の探索） |
|---|---|
| 「ここで辞めたら後悔するよ」「もう少しがんばろう」 | 「辞めたいって、ちゃんと言葉にできて偉いね（是認）。辞めたい気持ちと、まだ続けてここにいる気持ち、両方ある感じ？　続けてる側にある気持ちって、どんなところからきてる？」 |

### D. 「親に言われてやってる」と言われたとき

| ❌ NG | ✅ OK（焦点化＋自己決定強化） |
|---|---|
| 「親のために頑張ろう」「親孝行だと思って」 | 「親に言われて来てるんだね（反映）。じゃあ、君自身は『これだけは自分のためにやってもいいかな』と思えるものってある？　無くてもいい、ゼロでも教えて。」 |

### E. 「先生は分からない」と言われたとき

| ❌ NG | ✅ OK（自己一致＋I-メッセージ） |
|---|---|
| 「いや先生も昔は…」（経験談で奪わない） | 「分からない、って言ってくれてありがとう。確かに僕が分かりきれないところはたくさんあると思う（自己一致）。だから、君が一番分かってほしいところを一つ教えてほしいんだ（I-メッセージ＋焦点化質問）」 |

### F. 保護者からの強い要望に対応するとき

| ❌ NG | ✅ OK（受容→事実確認→選択肢） |
|---|---|
| 「規則ですので」「他のご家庭は…」（理屈・比較で封じる） | ①「お気持ち、教えてくださってありがとうございます。〇〇という点でご心配なんですね（要約）」②「事実関係を私からも一度確認させてください（時間を区切る）」③「対応として、A・B・Cの方向が考えられます。お家ではどれが現実的でしょうか（合意形成）」 |

### G. 三者面談で生徒の主体性を保つ

```
開始時の枠組み（オープン質問で生徒先発）：
「今日はまず〇〇さん本人の言葉から聞かせてもらっていい？
最近、勉強で『これはちょっと進んだな』と思えたこと、一つだけ。」

保護者の発言が長くなったら：
「お母さんの心配、わかりました（要約）。
〇〇さん本人は、今の話を聞いてどう感じた？」
```

### H. 質問しても黙ってしまうとき

```
①Wait-Time 3秒以上待つ
②「言葉にしづらいテーマかもね」とラベリング
③クローズドに切替：「YES/NOで答えていい？　いま、答えたくない気分？」
```

## 12. ⭐ 自己評価ルーブリック（MITI 3.0 日本語版簡略化）

講師が自身の対話を振り返るためのチェック項目：

- [ ] 開かれた質問の比率：少なくとも50%以上
- [ ] **反映：質問比 ≧ 2:1**（質問より反映を多く）
- [ ] 是認の頻度：1セッションで3回以上
- [ ] Change Talk を強化したか（Sustain Talkに正論反論していないか）
- [ ] 助言の早すぎる導入はないか

## 明日からできる3つのこと

1. **質問後に「3秒数える」習慣を始める**（Wait-Time意識づけ）
2. **「Why（なぜ）」を1日封印し、「どんな時」「何が」「どうしたら」に置き換える**
3. **You-メッセージをI-メッセージに変える練習（5回／日）**

## チェックリスト

- [ ] MI のOARS（4技法）を覚えている
- [ ] Change Talk と Sustain Talk を区別できる
- [ ] Rogers の3条件を説明できる
- [ ] I-メッセージの3要素構文を使える
- [ ] GROW 4段階を順に使える
- [ ] SFAの4技法（例外・ミラクル・スケーリング・コーピング）を区別できる
- [ ] Wait-Time 3秒以上を実践している
- [ ] メラビアンの法則の誤用を指摘できる
