# M10: トラウマインフォームドケアと学習トラウマ

## 学習目標

- SAMHSA 6原則・4Rs を教育場面に応用できる
- 学習トラウマの蓄積メカニズムと二次障害リスクを理解する
- TALK の原則・専門機関接続フローを獲得する
- 講師個人がカウンセラー役を担わないことの重要性を理解する

## キーポイント

1. **TIC（Trauma-Informed Care）**：「トラウマがあるかもしれない」観点で対応する支援の枠組み
2. **SAMHSA 6原則**：安全・信頼・ピア・協働・エンパワメント・文化配慮
3. **学習トラウマ**は不登校・抑うつ・希死念慮の主要因
4. **「責めない・比較しない・強制しない」3原則**
5. **TALK の原則**：希死念慮を示唆された時の緊急対応
6. **講師は「治療者」ではなく「安全な学習空間の提供者」**

## 1. トラウマインフォームドケア（TIC）基礎

### 1.1 TICとは

> トラウマがあるかもしれないという観点をもって対応する支援の枠組み

- 医療、教育、福祉、保健、地域で、こころのケガをした人に関わるすべての人が対象
- 米国SAMHSA（薬物乱用精神保健管理局）が体系化
- 日本では兵庫県こころのケアセンター（亀岡智美ら）が導入推進

### 1.2 ⭐ SAMHSA 6原則（教育場面適用）

| 原則 | 教育場面への適用 |
|---|---|
| 1. **Safety（安全）** | 怒鳴らない・人前で晒し上げない・物理的安全 |
| 2. **Trustworthiness & Transparency（信頼性と透明性）** | ルール明示、決めたことは覆さない |
| 3. **Peer Support（ピアサポート）** | 同じ困難を持つ仲間との繋がり |
| 4. **Collaboration & Mutuality（協働と相互性）** | 学習方法を一緒に決める、講師も完璧ではないと伝える |
| 5. **Empowerment, Voice & Choice（エンパワメント・声・選択）** | 教材形式・休憩を本人が選ぶ |
| 6. **Cultural, Historical & Gender Issues** | 個別文脈への配慮 |

### 1.3 4つのR

| R | 内容 |
|---|---|
| **Realize**（理解する） | トラウマが広範に存在し、回復の道筋があることを理解 |
| **Recognize**（気づく） | 本人・家族・スタッフのトラウマ反応の徴候に気づく |
| **Respond**（対応する) | 知識を方針・手順・実践に統合し、TIC原則で対応 |
| **Resist re-traumatization**（再受傷させない） | 政策・実践によって再びトラウマを与えないよう抵抗 |

### 1.4 日本の主要推進者

| 研究者 | 所属・著作 |
|---|---|
| **亀岡智美** | 兵庫県こころのケアセンター副センター長・児童青年期精神科医。『子ども虐待とトラウマケア』金剛出版 |
| **野坂祐子** | 大阪大学。『トラウマインフォームドケア』日本評論社、JST-RISTEXプロジェクトリーダー |
| **白川美也子** | 『赤ずきんとオオカミのトラウマ・ケア』、『子どものトラウマがよくわかる本』 |
| **藤森和美** | 学校トラウマ研究の中心。誠信書房の実践書多数 |
| **卜部明** | 『学校におけるトラウマ・インフォームド・ケア』遠見書房 |

## 2. 教育的トラウマの概念

### 2.1 学校で生じるトラウマ

藤森和美らが体系化：
- 事件・事故・災害
- **いじめ**
- **教師の不適切指導**
- **体罰**
- **性暴力**

学校場面で発生・累積するトラウマ。

国際的にはLee-Anne Gray博士の "Educational Trauma: Examples From Testing to the School-to-Prison Pipeline"（Springer, 2019）が体系化。

### 2.2 LD・発達障害児への蓄積効果

- 発達障害そのもの（一次障害）ではなく、**周囲とのミスマッチ・失敗体験・誤解・叱責の積み重ね**による二次障害が、不登校・抑うつ・引きこもりの主要因
- **不登校児の57%がASDやADHD等の発達障害**との調査結果

### 2.3 ACEs研究（小児期逆境体験）

- Felitti & Andaらが1990年代に開始
- 18歳までの逆境体験（虐待・ネグレクト・家族機能不全等）が累積するほど、成人期の精神・身体・社会的問題リスクが上昇
- 日本では兵庫県こころのケアセンターによる調査でも確認

## 3. ⭐ ディスレクシア・LD当事者の学習トラウマ

### 3.1 主因（蓄積的トラウマ）

ディスレクシア当事者の二次的トラウマの主因：

- 「もっと頑張れ」「努力不足」と誤解された叱責
- 音読・板書写しの強制による恥辱体験
- 同級生からの「バカ」「サボり」とのレッテル貼り
- 試験での連続失敗体験
- 親からの過剰な期待・落胆

「自分は本当は頭が悪くないのに、みんなは自分のことをバカだと思っている」と感じることで抑うつ状態や他者評価への過敏が形成される。

### 3.2 統計（不登校・二次障害）

- **LDの有病率**：世界5〜15%、日本のディスレクシアは6〜7%（NPO EDGE は約8%）
- **鳥取県調査**：**不登校児童の約35%、不登校生徒の約60%が学習障害**
- 発達性ディスレクシア児童生徒の不登校理由：「学習場面」50%、「対人場面」30%
- LD児は不安障害・抑うつ障害が高率に併存

## 4. 発達性トラウマ・複雑性PTSD

### 4.1 概念
- **複雑性PTSD（C-PTSD）**：Judith Herman（1992年『心的外傷と回復』）が提唱。ICD-11で正式採用
- **発達性トラウマ障害（DTD）**：Bessel van der Kolkらが提唱。子ども版C-PTSD
- **杉山登志郎**：日本で「第4の発達障害」としてトラウマ系発達障害を提唱

### 4.2 ⭐ Herman の3段階モデル

| 段階 | 内容 | 講師の関わり |
|---|---|---|
| **第1段階：安全の確立** | 物理的・心理的安全感の確保 | **塾講師・教員はここに徹する** |
| **第2段階：想起と服喪追悼** | 過去のトラウマを思い出して語る | **専門家領域**。講師が踏み込まない |
| **第3段階：再結合** | 新たな自己・人生の再構築 | 専門家との協働 |

→ 講師は**第1段階（安全な学習環境の提供）に徹し、必要に応じて専門機関へ繋ぐ**姿勢が重要。

## 5. ⭐ 「傷を再開させない」3原則

### 5.1 3原則

1. **責めない**：「努力不足」「さぼり」「やる気がない」と即断しない
2. **比較しない**：兄弟・他生徒との比較は厳禁。「Aさんはできたよ」NG
3. **強制しない**：音読・板書写し・人前発表など、過去にトラウマ化した活動を強制しない

### 5.2 安全な失敗体験の作り方

- 達成可能な小さな課題から始める（スモールステップ）
- **「間違える前提」のテスト設計**（白紙でもよい・部分点を多く）
- 失敗時に「教えてくれてありがとう」と返す
- 答えがわからない時の**「パス」「ヘルプ」コード**を用意

### 5.3 トリガー警告

- 急に音読を当てない（事前に「次の段落、読んでもらえる？難しければパスでOK」）
- 過去にトラウマ化した教材形式を回避（赤ペン大量添削、漢字テスト等）
- **教室前方での晒し上げ・公開叱責は絶対回避**

## 6. レジリエンス研究

### 6.1 Resilience の定義
- 困難な状況に適応する、抵抗する力。柔軟性・適応力・回復力
- 先天的資質要因と後天的に獲得可能な要因に分けて研究

### 6.2 Tedeschi & Calhoun のPTG（心的外傷後成長）5領域

心的外傷後成長尺度（PTGI）の5領域：

1. **他者との関係**（Relating to Others）：他者への思いやり、頼ってよい認識
2. **新たな可能性**（New Possibilities）：以前と異なる人生の方向性
3. **人間としての強さ**（Personal Strength）：自己の強さへの気づき
4. **精神性的・実存的変化**（Spiritual Change）
5. **人生への感謝**（Appreciation of Life）：優先順位の変化

### 6.3 日本での研究
- **平野真理**（東京家政大）：『レジリエンスは身につけられるか』、二次元レジリエンス要因尺度（資質的／獲得的）
- **佐藤健二**（徳島大）：トラウマ・PTSD・CBTの臨床研究

## 7. 不登校・引きこもり経験者への対応

### 7.1 多様な学びの場

| 学びの場 | 内容 |
|---|---|
| **N高・S高・N中等部・S中等部** | 角川ドワンゴ学園、オンライン学習。不登校受け皿として大きな役割 |
| **NIJINアカデミー** | 不登校小中学生向けオンラインフリースクール |
| **ROCKET → LEARN** | 東京大学先端研・日本財団、不登校・特異な才能を持つ子向け |
| **東京都 多様な学びの場ナビ** | 自治体ポータル |

### 7.2 復帰時の伴走原則
- **「学校に戻ること」をゴールにしない**（学習権の保障が本質）
- 朝起きられない・体調不良を「甘え」と判断しない（起立性調節障害等の身体症状を伴うことが多い）
- まずは1日10分の学習からなど、極端なスモールステップ
- **評価対象から外す配慮**（「テストしない」と明言）

## 8. 二次障害の早期発見

### 8.1 抑うつ・不安スクリーニング
- **K6（Kessler-6）**：6項目、10点以上で重篤な精神健康問題の可能性
- **SDQ**：子ども・青年向け、5下位尺度（多動/不注意・行為問題・情緒症状・仲間関係・向社会性）
- **UCLA心的外傷後ストレス障害インデックス**：兵庫県こころのケアセンターが日本語版整備

### 8.2 ⭐ 自殺リスクサイン（文科省「教師が知っておきたい子どもの自殺予防」H21）

#### 警戒サイン
- 自殺をほのめかす言葉（「死にたい」「いなくなりたい」「楽になりたい」）
- 大切な持ち物を譲る、別れの挨拶
- 自傷行為の痕跡
- 急な成績低下、不登校、人間関係からの撤退
- 睡眠・食欲の極端な変化
- 物質乱用、危険な行動

### 8.3 ⭐ TALK の原則（緊急時手順）

| 文字 | 内容 | 具体例 |
|---|---|---|
| **T**ell | 言葉で伝える | 「あなたのことが心配です」 |
| **A**sk | 率直に尋ねる | 「死にたいと思うことがある？」（タブー視せず聞いてよい） |
| **L**isten | 傾聴する | 絶望感を「そう思ってしまうんだね」と評価せず傾聴。**説教・励まし・解決提案は厳禁** |
| **K**eep safe | 安全確保 | 一人にせず、専門家に繋ぐ |

## 9. 講師・教員のセルフケア

### 9.1 Compassion Fatigue（共感疲労）
- トラウマを抱える人への支援に従事する人の**約4分の1**が経験
- 看護師・介護職・カウンセラー・教員に多い

### 9.2 Vicarious Trauma（代理受傷／二次受傷）
- 被害者の話を熱心に聴くうちに、被害者と同じような心身反応（フラッシュバック様症状・睡眠障害・回避等）が起こる現象
- 法政大学・廣川進の警察庁向け教材が標準的解説資料

### 9.3 バーンアウト予防
- **業務外時間の境界線**の保持（バウンダリー）
- **一人で抱え込まず**、複数の同僚・スーパーバイザーで共有
- マインドフルネス・身体的セルフケア
- **自身も支援を受ける**（カウンセリング、ピアグループ）

詳細は [[TL_M12-Teacher-Self-Care]]

## 10. ピアサポートと当事者コミュニティ

### 10.1 ピアサポートの治癒力
- 「Peer（仲間）」と「Support（支援）」の合成
- 共通項を持つ人同士の支え合いが、専門職支援とは異なる固有の治癒力を持つ
- 「同じ経験をした人がいる」「自分だけじゃない」という認識が、孤立感・スティグマを軽減

### 10.2 ディスレクシア当事者コミュニティ
- **NPO EDGE「DX会」**：成人ディスレクシア当事者の会
- **触るグリフ**：当事者体験談の集積・発信
- **発達性ディスレクシア研究会**：当事者・家族・支援者ネットワーク

## 11. ⭐ 実装可能な対話スクリプト集

### 11.1 過去のトラウマ体験を話された時

**生徒**：「前の塾で『お前は本当にバカだな』って言われて、それから国語が怖くて……」

**❌ NG**：
- 「そんな先生は最低だね！」（加害者評価を即決しない）
- 「もう過去のことだよ、忘れよう」（軽視）

**✅ OK**：
```
「話してくれてありがとう。それはとても辛かったね」（感謝＋感情の言語化）
「言葉で傷つけられた体験は、本当にこころに残るよね」（傷の正当化）
「ここでは絶対にそういうことは言わない、と約束するね」（安全の保証）
「もし読むのが怖い時は『パス』って言ってくれていいよ」（具体的な選択肢）
```

※深く掘り下げず、想起作業に踏み込まない。必要に応じてスクールカウンセラー・専門機関への接続を提案。

### 11.2 「先生はバカにしてないって本当？」と疑われた時

**❌ NG**：「そんなことないよ！」（即否定は信用されない）

**✅ OK**：
```
「そう思わせてしまうような態度があったら、教えてほしい」（謙虚に問う）
「これまで何人もの先生に同じこと言われて、信じられなくなって当然だと思う」（疑念の正当化）
「だから、僕の言葉じゃなくて、これからの僕の行動で判断してほしい」（時間軸で示す）
「君のディスレクシアのことを勉強してきた。読めない・書けないは『頭が悪い』とはまったく違う、というのが今の科学の答えなんだ」（事実情報）
```

### 11.3 自己肯定感が低い生徒への声かけ

**❌ NG**：「みんな頑張ってるんだから」「そんなこと言わないで」「もっと自分に自信を持って」

**✅ OK の枠組み**：
- **具体的事実を返す**：「今日は最初の3問、自分で解けたね」
- **プロセスを認める**：「答え合ってたかどうかより、こうやって考えた道筋がすごい」
- **比較しない**：「前の自分」とだけ比べる（「先週はここで詰まってたよね」）
- **強みを別領域から探す**：「君、絵うまいよね」「人の話よく覚えてるよね」

### 11.4 不登校から復帰する生徒への声かけ

**初回登塾時**：
```
「来てくれてありがとう」（学校復帰でなく「来た事実」を歓迎）
「今日は何分くらいなら大丈夫そう？10分でも全然いいよ」（本人に選択権）
「途中で帰っていいし、横になりたかったら言って」（撤退ルートの保証）
「テストやドリルは今日はやらない、お話だけしよう」（評価圧の除去）
```

**継続時**：
- 朝起きられない等の身体症状を「甘え」と捉えない
- 「学校に戻る」を目標にしない／「学力をつける」を当面の目標にも置きすぎない
- 1ヶ月、3ヶ月、半年単位の超長期視点

### 11.5 希死念慮を示唆された時の対応（緊急時手順）

#### サインに気づいたら（「死にたい」「消えたい」「もう疲れた」等）

**Step 1: その場での対応（TALK の原則）**
- **T（Tell）**：「あなたのことが心配です。今、すごく辛い状態なんだね」
- **A（Ask）**：「『死にたい』って気持ちが今あるの？」と直接尋ねる（タブー視せず聞いてよい）
- **L（Listen）**：絶望感を「そう思ってしまうんだね」と評価せず傾聴。**説教・励まし・解決提案は厳禁**
- **K（Keep safe）**：「一人にしない」「具体的な手段の準備があるか確認」

**Step 2: 即時連絡**
- 保護者への連絡（ただし家庭が虐待加害源の場合は児童相談所優先）
- 24時間子供SOSダイヤル：**0120-0-78310**
- 緊急性が高ければ救急要請（119）／警察（110）

**Step 3: 専門機関接続**
- スクールカウンセラー、児童精神科、地域の精神保健福祉センター
- 兵庫県こころのケアセンター等のトラウマ専門機関
- 児童相談所：**189（いちはやく）**

**Step 4: 講師自身のケア**
- **一人で抱え込まない**（守秘義務と専門機関連携のバランス）
- 経過は記録に残す
- 自身も代理受傷リスクがあるため**スーパーバイズを受ける**

## 12. ⭐ 重要な姿勢（明示すべきこと）

> **講師個人がカウンセラー役を担うのではなく、専門機関と連携する**

- 講師は「治療者」ではなく**「安全な学習空間の提供者」**
- 心理的に深い領域は児童精神科・心療内科・公認心理師等の専門家へ
- 「想起作業（過去のトラウマを思い出して語る）」を講師主導で行わない
- 自殺リスク等は組織として速やかに専門機関へ繋ぐ
