# M2: 強みベースで関わる

## 学習目標

- "Dyslexic Thinking is a Skill" の意味を理解する
- 「治す／直す」フレーミングがなぜ害をなすか説明できる
- 強みベースの声かけを日常的に実践できる

## キーポイント

1. **「治す」ではなく「活かす」**（NPO EDGE / Made by Dyslexia 共通哲学）
2. 「がんばれば読める」は**最大のトラウマ**になる
3. **6つの Dyslexic Thinking Skills** を知る
4. 声かけは**事実ベース・選択ベース**で
5. AI時代に求められる知性こそディスレクシック・シンキング

## 1. なぜ「強みベース」なのか

### 弱点補正モデルの限界
- 「読みが弱い → 読みの練習をたくさん」式の補正は、本人の自己肯定感を傷つけやすい
- 「直そうとされている」感覚が「自分は壊れている」自己認識を強化
- 補正で得られる読みの改善はわずか、その間に自己効力感が大きく毀損

### 強みベースのアプローチ
- 本人の**得意な認知特性**（視覚的・全体的思考など）を活かす
- 苦手な部分は**ツールで代替**（読み上げ、OCR、ふりがな等）
- 「**違う学び方をしている**」と捉え、その学び方に合う環境を整える

## 2. NPO EDGE 藤堂栄子氏の哲学

> 「ディスレクシアは**多様性**。適切な支援があれば素晴らしい才能が開花する」

- スローガン：「ディスレクシア（読み書き困難）が**活きる**社会」
- **「治す」ではなく「活かす」**フレーミング
- 当事者を**運動の主体**として位置づける

詳細は [[V_NPO-EDGE]]

## 3. Made by Dyslexia "Dyslexic Thinking is a Skill"

### 共通スローガン："Spot, Support, Empower"
- **Spot**（見つける）：特性を早期に発見
- **Support**（支える）：適切な配慮を提供
- **Empower**（力を引き出す）：強みを伸ばす

「治す（Cure）」「直す（Fix）」「克服する（Overcome）」は使わない。

### LinkedIn / Dictionary.com にも認定
- LinkedIn は2022年に "Dyslexic Thinking" を**公式スキル**として追加
- Dictionary.com にも収録：能力として認知されつつある

### Intelligence 5.0 レポート（2024）
- Randstad との共同調査（米英印豪 5,000人 YouGov）
- 「複雑な問題解決」を最重要知性と考える人 **73%**
- 「標準化された試験こそ知性」と考える人わずか **6%**
- → **「AI時代に求められる知性はディスレクシック・シンキングそのもの」**

詳細は [[V_Made-by-Dyslexia]]

## 4. ⭐ 6つの Dyslexic Thinking Skills

| スキル | 意味 | 統計（ディスレクシア当事者のうち） |
|---|---|---|
| **Visualising** | 視覚的・空間的に物事を捉える | 84% が平均以上 |
| **Imagining** | 新しい発想・独自の切り口 | 71% が平均以上 |
| **Communicating** | 人を引き込む話し方・表現 | 84% が平均以上 |
| **Reasoning** | パターン・可能性を見抜く | — |
| **Connecting** | 点と点を結んで新しい意味を作る | — |
| **Exploring** | 好奇心をもって深く掘り下げる | — |

教室で意識的に**これらの強みを発揮できる機会**を作る：

- 視覚的なまとめ・マインドマップ
- グループでアイデアを出す活動
- 口頭発表・対話
- パターン発見型の問い
- 教科横断的な探究

## 5. 声かけの原則（実践編）

### NG → OK 変換表

| ❌ NG | ✅ OK |
|---|---|
| もっとがんばろう | 別の方法を試してみよう |
| 何度でも練習しよう | こういう道具もあるよ |
| 集中して読みなさい | ゆっくりで大丈夫 |
| 正しく読めましたか？ | ここまで読めたね |
| みんなと同じように | あなたなりのやり方で |
| 障害を克服しよう | 自分に合う環境を作ろう |
| 苦手を直そう | 得意を活かそう |
| 読めないのは努力不足 | 読み方は人それぞれ |

### 「事実ベース」の声かけ
- 評価語（すごい・素晴らしい・正解）を使わず、**事実**を述べる
- 「今日は20分集中できたね」
- 「最後まで読めたね」
- 「設定が見つかったね」

### 「選択ベース」の声かけ
- 「やる／やらない」ではなく「**どれを試す？**」
- 「読む／読まない」ではなく「**どの方法で読む？**」（音読・読み上げ・分かち書き等）

## 明日からできる3つのこと

1. **「がんばろう」「努力しよう」を口にしない24時間を作ってみる**
2. **「あなたの〇〇な発想がよかった」と強みを具体的に言葉にする**
3. **「合う方法を一緒に探そう」を口癖にする**
