# Curriculum-Outline — 講師向け学習カリキュラム全体構成（M1〜M12）

> アプリを使う**講師（学習塾講師・学校教員・保護者・支援員）**が、ディスレクシアのエビデンスと実践を**段階的に学べる**コンテンツ。
> Phase 1-A／1-B（学術・現場・当事者）に加え、**Phase 1-C（心理学根拠）**を統合した12モジュール構成。
> アプリ内ヘルプ／オンライン公開／PDF配布資料いずれにも転用可能。

## カリキュラム設計方針

1. **エビデンスベース**：すべての主張に査読論文・公的ガイドラインの出典を明示
2. **実践重視**：「明日から教室／家庭でできる」レベルに具体化
3. **強みベース**：「弱点の補正」ではなく「特性を活かす」視点
4. **心理学的に裏付けされた声かけ**：SDT・Dweck・Bandura・MI・SAMHSAの統合
5. **段階的**：M1（基礎）→ M12（セルフケア）と理解を深める
6. **短時間モジュール**：各モジュール15〜30分で読了可能

## 全体の学習目標

修了後、以下ができることを目指す：

### 基礎（M1-M6）
- [ ] ディスレクシアを**「治療すべき病気」ではなく「学び方の違い」**として理解する
- [ ] 「がんばれば読める」式の励ましがなぜ傷つけるかを言語化できる
- [ ] 個人差を前提に**「複数の方法から本人が選ぶ」**支援観を持つ
- [ ] 査読論文の主要知見と公的ガイドラインの数値を引用できる
- [ ] 教室・家庭での具体的な配慮を3つ以上実践できる
- [ ] 合理的配慮の申請プロセスを生徒・保護者と協働で進められる

### 応用（M7-M12）★Phase 1-C
- [ ] **中田螺旋形モデル**で保護者面談の6ステップを実践できる
- [ ] **エクスプレッシブ・ライティング等のエビデンスベース介入**を受験生に提案できる
- [ ] **False Growth Mindset の罠**を回避し、戦略ほめ・工夫ほめを実践できる
- [ ] **SAMHSA 6原則**で「再受傷させない」教室を設計できる
- [ ] **TALK の原則**で希死念慮を示唆された時の緊急対応ができる
- [ ] **MI・OARS・Rogers・I-メッセージ・GROW・SFA**を場面別に使い分けられる
- [ ] **Wait-Time 3秒以上**を意識して実践している
- [ ] **共感疲労・代理受傷・バーンアウト**を自己モニタリングできる

## 推奨する学習順序

### 標準ルート（時間に余裕のある場合）
```
M1 → M2 → M3 → M4 → M5 → M6 → M7 → M8 → M9 → M10 → M11 → M12
（基礎12週間）
```

### 緊急時最優先ルート（傷つけない関わり方を最初に）
```
M2（強みベース）→ M7（告知）→ M10（トラウマ）→ M11（対話）→ M12（セルフケア）→ 残り
```

### エビデンス重視ルート
```
M1 → M3 → M9（動機づけ理論）→ M4 → M8（受験生）→ 残り
```

### 受験塾向けルート（塾長・予備校講師）
```
M2 → M8（受験生）→ M9（動機づけ）→ M11（対話）→ M10（リスク対応）→ M12（セルフケア）→ 残り
```

## 各モジュールの構成（共通）

すべてのモジュールは以下のセクションで構成：

1. **学習目標**
2. **キーポイント（要約）**
3. **本論**（複数のサブセクション）
4. **明日からできる3つのこと**
5. **チェックリスト**
6. **参考文献・関連リンク**
