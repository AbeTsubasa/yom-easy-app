# M7: 検知時の告知・初期対話の心理学

## 学習目標

- 中田洋二郎の螺旋形モデルを理解する
- 本人・保護者への告知の年齢別・文化的配慮を獲得する
- 塾講師として「医師でない立場」で適切な言葉を選べる

## キーポイント

1. **ディスレクシアは「確定困難型」障害** → 螺旋形モデル（中田 1995）が適合
2. 受容は**ゴールではなく連続的プロセス**。進級・受験期に再起する
3. **「ねぎらい」を先にする保護者面談**（6ステップ）
4. **塾講師は「可能性」を示すのが業務範囲**（医師法上の制約）
5. **「学び方の違い」というフレーミング**が最も中立で実装的

## 1. 中田洋二郎の螺旋形モデル（1995）

### 先行モデル

#### Drotar段階説（1975）
ショック → 否認 → 悲しみと怒り → 適応 → 再起の5段階モデル。

**段階説の限界**：「再適応」を前提とするため、達していない親に過酷な要求になりうる。

#### Olshanskyの慢性的悲哀（1962）
精神遅滞児の親の悲哀は終結せず、**「発達の節目（歩行、就学、進学）に再起する周期的反応」**。

### 中田の統合モデル

> **「肯定と否定の感情が表裏一体として常に存在する螺旋形」**

- 受容は「到達すべきゴール」ではなく**「連続的な過程」**
- 否認に見える行動（医療機関ショッピング等）も裏側に「現実を認めようとする葛藤」
- **確定困難型障害（自閉症・LD・ディスレクシア）ほど慢性的ジレンマ型**

### ディスレクシアへの含意
- **「治る／受け入れる」式の単純な進歩モデルを当てはめない**
- 進級・受験期に保護者の悲哀が再起することを予測し、節目ごとに面談を重ねる
- 「もう受け入れたはず」と思った保護者が、進学のたびに揺り戻すのは自然

## 2. 本人への告知（年齢別）

国立障害者リハビリテーションセンター3原則：
1. **本人にとってのタイミング**で
2. **本人がわかる言葉**で
3. **本人が納得できる説明**を

### 2.1 小学生

#### 認知発達上の押さえどころ
- 他者比較が始まるのは概ね**10歳前後**（ピアジェ的具体的操作期後期）
- それ以前は「自分は悪い子」と漠然と思いやすい
- 抽象ラベルの意味が掴めず、「君は〇〇です」式は**「自分=悪い」固定的同一視**を生む

#### 推奨メッセージ
- **「苦手なこと（短所）は別の方向から見れば強み（長所）でもある」**
- 「これらの特徴は脳に関連した生まれつきの性質。成長や経験で変化していくもの」
- 「同じ診断名でも一人ひとり性格や個性が違う」
- 「診断がある人とない人とははっきり区別できない」

#### 補助
- 口頭だけでなく、絵や図、メモを併用
- 「周囲に困っている人がいれば手助けするのと同じように、自分の苦手を周りに伝えてよい」

### 2.2 中学生・高校生

#### アイデンティティ形成期の配慮
- 思春期は隠しながらうまく過ごす力もつく反面、人知れずストレスを抱える
- 「障害＝病気」ではなく**「特徴」「特性」**と伝える
- 自尊心・自己肯定感を傷つけない言葉を選ぶ

#### セルフアドボカシーへの橋渡し
- セルフアドボカシー＝自分の権利を自分で主張する力
- 周囲が代弁し続けると、本人が発信する機会と力を奪う
- 中学生以降は「**合理的配慮を自分で申請する**」練習段階へ

#### 受験生への影響
- 大学入学共通テスト・高校入試での合理的配慮（時間延長・別室・ルビ振り）申請には診断書が必要なケースがある
- **配慮申請時期から逆算した早期告知**（小5〜中1あたり）が望ましいケースが多い

### 2.3 成人（後天的告知）
- 「この苦しさを誰にもわかってもらえない」と長年悩み続け、自ら受診する例
- 「もっと早く知りたかった」「人生の説明がついた」両方の反応
- **「腑に落ちる」体験**を支援する

## 3. 保護者への告知 — 6ステップ

### 3.1 ⭐ 6ステップ面談プロトコル（中田＋SPIKES統合）

#### ステップ1：場の設定（Setting）
- 個室、十分な時間（最低40分）
- 第三者の同席可否を保護者に選ばせる

#### ステップ2：これまでの認識を聴く（Perception）
- 「いつ頃から気になっていましたか」「ご家庭ではどう見えていますか」
- **まず保護者の認識から始める**

#### ステップ3：ねぎらい・正常化 ★日本文化に必須
- 「ここまでよく見てこられましたね」
- 「気づくことが第一歩です」
- 自責の念をほぐす

#### ステップ4：客観情報の共有（Knowledge）
- 観察事実をデータで示す（誤読のパターン、作文サンプル等）
- **診断名は使わず「可能性」として提示**

#### ステップ5：感情への対応（Emotion）
- 「驚かれましたよね」「ショックですよね」と感情を言語化して受容

#### ステップ6：次の一歩の合意形成（Strategy）
- 専門機関の橋渡し、家庭でできる工夫、次回面談日程

### 3.2 父母の反応パターンの違い
- Damrosch & Perry（1989）：慢性的悲哀の経験率は**父17%、母64%**
- 母親は日常の困難に直面しやすく感情的反応が前面に
- 父親は「治る方法は?」と問題解決モードに入ることが多い

→ **両親同席が理想**だが、参加しない側へのフォロー（書面サマリー、別途面談）を構造化。

### 3.3 フォローアップ
- **48時間以内**：保護者にメールで「今日はありがとうございました。お話ししたポイントを書面でお送りします」
- **1週間後**：受容過程の進み具合を確認
- **1ヶ月後**：専門機関受診の進捗、家庭での工夫の効果共有
- **学期ごと**：成長のエビデンスを共有 — 慢性的悲哀の再起時期（進級・受験期）には特に厚く

## 4. 日本特有の文化的配慮

| 文化要素 | 対話戦略 |
|---|---|
| **「迷惑をかけない」志向** | **「眼鏡をかけるのと同じ」比喩**で説明 |
| **「がんばる」文化** | 「努力の方向を変える」フレーミング |
| **スティグマ** | 守秘・プライバシー徹底 |
| **家族主義** | **祖父母向け説明資料**を副資料で用意 |

## 5. 塾講師・教員の立場の制約

### 5.1 絶対に避けるべき表現
- 「お子さんはディスレクシアです」「LDですね」
- 医師法上の制約 + 誤診回避

### 5.2 適切な表現
- 「読み書きについて、特性に応じた支援が役立つお子さんかもしれません」
- 「一度専門機関でアセスメントを受けると、お子さんに合った学び方が見えてきます」
- 「学習スタイルの違いがあるようなので、得意な感覚チャンネルを活かす方法を一緒に考えませんか」

### 5.3 専門機関への橋渡し
- 第一窓口：スクールカウンセラー、特別支援教育コーディネーター、教育委員会教育相談窓口
- 第二窓口：児童精神科、小児神経科、発達障害者支援センター、各都道府県のLD協会
- 専門アセスメント機関：発達性ディスレクシア研究会、NPO法人LD・Dyslexiaセンター

## 6. フレーミング論：用語の使い分け

| 用語 | 使う場面 | 注意点 |
|---|---|---|
| 障害 | 制度・診断書・合理的配慮の根拠説明 | 「治す」「克服」とセットにしない |
| 特性 | 本人・保護者との日常対話 | 個性論への矮小化に注意 |
| 個性 | 強みを伸ばす文脈 | 困難を軽視するリスク |
| **学び方の違い / 学習スタイル** | 学習方法を提案する時 | **最も中立で実装的** |
| 読み書きの困りごと | 機能的・操作的に話す時 | 障害概念に巻き込まれない |

→ **「学び方の違い」「読み書きの困りごと」「特性」を主軸**にし、「障害」は制度文脈で限定使用。

## 7. ⭐ 対話スクリプト集

### 7.1 本人への初期対話（小学校中学年〜）

```
塾講師：◯◯くん、最近の音読、すごく頑張っているね。
　　　先生、ちょっと気づいたことがあるんだけど、聞いてもいい？

生徒：うん。

塾講師：◯◯くんは、たくさんのことを覚えるのが得意だよね（強みから始める）。
　　　でも、文字を読むときに目がすごく疲れたり、行を見失ったりすることがあるかな？

生徒：…うん、よくある。

塾講師：そうか、教えてくれてありがとう。
　　　実はね、人にはそれぞれ「学び方の違い」があってね、
　　　◯◯くんみたいに、頭はすごく動いてるけど、文字を読むときだけ脳が
　　　ちょっと違う働き方をする人が、クラスに2〜3人はいるんだよ。
　　　ぜんぜん珍しいことじゃないんだ。

生徒：…ふーん。

塾講師：これは「悪い」ことじゃないし、「治す」ものでもないんだ。
　　　眼鏡をかけてる人がいるみたいに、◯◯くんに合った「読み方の工夫」が
　　　いろいろあるから、一緒に試してみよう。
　　　◯◯くんはどう感じる？聞きたいことある？
```

### 7.2 保護者への初期対話（6ステップ実装）

```
【ステップ1：場の設定】
「お時間いただきありがとうございます。今日は40分ほどお取りしています。
　ご主人様にも後でお伝えしたいので、終わりにメモをお渡ししますね」

【ステップ2：これまでの認識を聴く】
「ご家庭で、◯◯くんの読み書きについて、何か気になることはありますか？」
（沈黙を恐れず、十分待つ）

【ステップ3：ねぎらい・正常化】
「これまで宿題のこと、毎晩横についてくださっていたんですね。
　本当に大変な思いをされてきたと思います。
　お母様が早く気づいてくださったから、今こうしてお話しできているんです」

【ステップ4：客観情報の共有】
「先月の音読チェックで、◯◯くんは初見の文章で行飛ばしが◯回、
　促音（っ）の読み飛ばしが◯回ありました。一方、内容理解は学年以上なんです。
　これは『読みの困りごと』のサインかもしれないと感じています。
　専門用語では『ディスレクシア（発達性読み書き障害）の可能性』と呼ばれる状態に
　近いかもしれません。ただ、これは医師の診断が必要な領域なので、
　私からは『可能性』としてお伝えしています」

【ステップ5：感情への対応】
「驚かれましたよね。お気持ちはいかがですか？」
（保護者の反応を否定せず受け止める）
「ショックを受けるのは自然な反応です。多くの保護者の方が同じ気持ちを
　経験されています」

【ステップ6：次の一歩の合意形成】
「お時間あるときに、◯◯（地域の専門機関）に相談されることをお勧めします。
　紹介状を書くこともできますし、塾でできる工夫もご相談したいです。
　今日決めなくて大丈夫です。一週間後にまたお話しできますか？」
```

## 明日からできる3つのこと

1. **「ディスレクシア」を「読み書きの特性」「学び方の違い」と呼ぶ言葉を1週間続けてみる**
2. **保護者面談で「ステップ3：ねぎらい」を必ず最初に入れる**
3. **「お子さんは〇〇です」式の断定を避け、「可能性」「特性」フレーミングを使う**
