# M6: 生徒・保護者との対話

## 学習目標

- 生徒の傷ついた経験に**傷を再開させない**言葉で対話できる
- 保護者面談での言葉選びを獲得する
- 「ディスレクシアの可能性」を初めて伝える場面で適切に振る舞える

## キーポイント

1. **「治す」「直す」「克服」を1度も使わない対話**を実現
2. 生徒・保護者の**過去の傷つき経験**を前提に
3. 「**選択肢**」と「**事実ベース**」で対話を組み立てる
4. 「**本人の声を最優先**」する姿勢
5. **強み視点**を保護者にも共有

## 1. 生徒との初期対話

### ⭐ やってはいけない初手

- 「君はディスレクシアの**可能性**がある」（医療フレーミング）
- 「特別な支援を**受ける必要**がある」（受動的・劣位）
- 「**がんばろう**」「**乗り越えよう**」（押し付け）
- 「クラスのみんなに**説明**しよう」（プライバシー侵害）

### 推奨される初手

```
「文字を読むのが苦手だなって感じる時、どんな時？」
「どんな見え方をしているのかな、教えてくれる？」
「読み方は人それぞれだから、君に合う方法を一緒に見つけよう」
```

### ⭐ 重要な前提

生徒は既に**深く傷ついている可能性が高い**：
- 「もっとがんばれば」と言われ続けた
- 「バカ」「読めないの？」と揶揄された
- 「努力不足」と評価された
- 自分でも「自分はバカなのかも」と思い込んでいる

→ **最初の対話で再び傷つけない**ことが何より大切

## 2. 生徒が「自分がディスレクシアらしい」と気づいた時

### 「説明してほしい」と言われたら

#### ❌ NG
- 「君の脳に問題があって…」
- 「治療法はないけれど、訓練で改善できる」
- 「世の中には君のような障害を持つ人がいて…」

#### ✅ OK
```
「文字を読むのが、ほかの人と違う仕組みで処理されているみたい。

仕組みが違うから、文字をそのまま読むより、
読み上げで聞いた方が頭に入ったり、
広い行間の方が読みやすかったりすることがあるんだ。

これは『脳の仕組みの違い』で、知能とは関係ないよ。

世界には、君と同じ仕組みの人がたくさんいて、
リチャード・ブランソンやスティーブン・スピルバーグも同じ仕組みを持っているらしい。

『治す』ものではなくて、自分に合う方法を見つけるものなんだ。」
```

### 「自分はバカなんですか」と聞かれたら

```
バカではないよ。

知能と読みは別の仕組みだから、
読みが苦手でも、ほかの力が強いことがよくある。

例えば、絵や図で考えるのが得意だったり、
独自のアイデアを思いついたり、
人と話すのが上手だったり——

そういう力を持っている人が多いって、
研究でも分かっている。
```

## 3. 保護者との初期対話

### ⭐ 保護者も傷ついている

- 「うちの子の育て方が悪かった」と自責
- 「もっと早く気づくべきだった」と後悔
- 親族・夫婦間で「育て方」をめぐる衝突
- 学校・友達からの孤立感

→ **保護者を責めない、ねぎらう**ことが第一歩

### 推奨される初手

```
「お子さんの読み書きについて、お話したいことがあります。

これまでお家でも色々な工夫をされてきたと思います。
ありがとうございます。

最近のお子さんの様子を見て、
『読み書きの認知の仕組み』が他の子と少し違うかもしれない、
と感じています。

これは『障害』というよりは『学び方の違い』で、
適切な方法に出会えれば、お子さんは本来の力を発揮できると思います。

一緒に方法を探していきませんか。」
```

### 「ディスレクシア」という言葉を出すタイミング

- いきなり医療用語を出さない
- まず「読み書きの認知の仕組みの違い」と説明
- 興味を示されたら「これは『ディスレクシア』と呼ばれることがあります」
- 「診断」「治療」ではなく「特性の理解」として

## 4. 保護者面談での進め方

### ステップ1：これまでの工夫を聴く
- 「お家でどんな工夫をされてきましたか？」
- 「お子さんはどんな様子ですか？」
- 否定せず、まず聴く

### ステップ2：観察した事実を共有
- 「学校では、こういう場面で時間がかかっています」
- 「漢字テストよりも、ノートを取る場面で困っているようです」
- 評価ではなく**事実**

### ステップ3：可能性を伝える
- 「読み書きの認知の仕組みが、他のお子さんと違う可能性があります」
- 「これは『学び方の違い』で、知能の問題ではありません」
- 著名当事者の例（Branson、ミッツ等）を**控えめに**

### ステップ4：強みを共有
- 「お子さんは、〇〇な発想が素晴らしいですね」
- 「絵を描くと、他のお子さんと違う視点を持っています」
- **事実ベース**で

### ステップ5：選択肢を示す
- 専門機関での相談（必要なら）
- 学校での合理的配慮
- 家庭でのICT活用
- 本アプリのような支援ツール
- **「やる／やらない」は保護者と本人が決める**

### ステップ6：継続的サポートの約束
- 「ご家庭と相談しながら進めましょう」
- 「気になることがあればいつでもご連絡ください」

## 5. 「専門機関」を勧める時の言葉選び

### ❌ NG
- 「**診断**を受けてください」（医療フレーミング）
- 「**治療**が必要かもしれません」
- 「**早期発見**が大切です」（焦らせる）

### ✅ OK
- 「お子さんの**特性をより詳しく知る**ためのアセスメントがあります」
- 「**選択肢を増やす**ために、専門家の意見を聞いてみるのもいいかもしれません」
- 「ご家族の判断で**ゆっくり考えて**ください」

### 紹介できる機関
- 大阪医科薬科大学LDセンター（→ [[F_Universities]]）
- 発達性ディスレクシア研究会会員（→ [[V_NPO-EDGE]]）
- 地域の特別支援教育センター
- NPO EDGE 相談窓口

## 6. 二次的トラウマへの対応

### サインに気づく
- 不登校・登校渋り
- 自己肯定感の低下（「どうせ自分なんて」）
- 攻撃的・反抗的行動
- 抑うつ・無気力
- 希死念慮（早急に対応）

### 関わり方
- **責めない**：「もっと努力していれば」式は絶対NG
- **ねぎらう**：「これまで本当に大変だったね」
- **可能性を示す**：「合う方法がきっとある」
- **専門機関と連携**：スクールカウンセラー、医療機関

詳細は [[V_Typical-Experiences]]

## 7. 本人の自己理解（セルフアドボカシー）を支える

→ [[F_UTokyo-Projects|DO-IT Japan]]の核心

- 「自分はどう学ぶのが合うか」を本人が言語化できるように
- 「私はディスレクシアで、こういう配慮が必要です」と本人が説明できる
- 大学・就職時に大きな武器になる

本アプリの設定プロファイルが、その**ツール**として機能する。

## 明日からできる3つのこと

1. **生徒・保護者との対話で「治す」「直す」「がんばろう」を一切使わない24時間を作る**
2. **「これまで本当に大変だったね」とねぎらう一言を入れる**
3. **強みを具体的に1つ挙げて事実ベースで伝える**
