# M9: 動機づけ理論の教育応用

## 学習目標

- SDT・Dweck・Banduraの3大理論を教育場面に応用できる
- **False Growth Mindset の罠**を理解し回避する
- 「ほめる／叱る」の科学的根拠に基づく実践を獲得する

## キーポイント

1. **SDT 3欲求**：自律性・有能感・関係性を満たす指導が深い学習を促す
2. **Dweckの2015年自己修正**：「努力すれば誰でもできる」は **False Growth Mindset**
3. ディスレクシアに対し**「努力で克服」式メッセージは学術的に有害**
4. **「努力ほめ」ではなく「戦略ほめ・工夫ほめ」**へ
5. **Banduraの結果期待 vs 効力期待**：両方を支援する必要

## 1. 自己決定理論（Self-Determination Theory, SDT）

### 1.1 提唱者と理論
- Edward L. Deci & Richard M. Ryan（1985年 *Intrinsic Motivation and Self-Determination in Human Behavior*）
- 先行する1971年のアンダーマイニング実験から発展

### 1.2 3つの基本的心理欲求

| 欲求 | 定義 | 教育場面での支援 |
|---|---|---|
| **自律性（Autonomy）** | 自分で選び、自分の意思で行動している感覚 | 選択肢の提供、決定権の委譲 |
| **有能感（Competence）** | 「できる」「上達している」感覚 | 適切な難度設定、進捗の可視化 |
| **関係性（Relatedness）** | 他者とつながり、認められている感覚 | 信頼関係、無条件の肯定 |

### 1.3 動機づけの内在化連続体

```
無動機
  ↓
外的調整（罰回避：「叱られないため」）
  ↓
取り入れ的調整（恥・プライド：「ダメな子と思われたくない」）
  ↓
同一化的調整（「自分にとって重要」と納得） ← 重要分岐点
  ↓
統合的調整（自己の一部）
  ↓
内発的動機づけ（楽しさ）
```

### 1.4 西村多久磨ら（2011）の重要知見

『教育心理学研究』59巻1号で、中学生対象に：
- **同一化的調整がメタ認知的方略を介して学業成績を予測**
- 純粋な内発的動機（楽しさだけ）は**必ずしも成績を予測しない**

→ アプリのメッセージは「楽しいよ」より**「あなたにとってこの読みやすさが大事」**を強調すべき。

### 1.5 自律性支援的 vs 統制的

| 自律性支援的 | 統制的 |
|---|---|
| 「どの問題から始めたい？」 | 「ちゃんとやらないとダメでしょ」 |
| 「この方法を選んだ理由を教えて」 | 「決まりだから」 |
| 「君がこう感じるのもわかるよ」 | 「みんなやってるよ」 |
| 「合わなかったら別の方法もある」 | 「これが正しいやり方」 |

Reeve et al. (2004)：自律性支援的な教師の下では深い学習・自己効力感が促される。

### 1.6 ディスレクシア指導への含意

- 「やらされ感」を最小化することが特に重要
- 失敗経験を多く抱える子は**「外的調整（叱られないため）」段階に留まりやすい**
- **関係性（信頼できる大人がそばにいる）と自律性（どう取り組むか選べる）を先に確保**しないと、有能感アプローチだけでは機能しない

## 2. マインドセット理論（Carol Dweck）

### 2.1 Fixed vs Growth Mindset

| Fixed Mindset（硬直） | Growth Mindset（成長） |
|---|---|
| 能力は生まれつきで変わらない | 能力は努力と戦略で伸ばせる |
| 困難を避ける | 困難を学習の機会と捉える |
| 失敗を能力の証明と捉える | 失敗を学びの機会と捉える |

### 2.2 Mueller & Dweck（1998）古典研究

小学5年生128名・Raven's Progressive Matrices：

| 群 | ほめ言葉 | 結果 |
|---|---|---|
| **能力ほめ** | "You must be smart" | パフォーマンス目標志向、失敗後に持続性低下、課題の楽しさ減 |
| **過程ほめ** | "You must have worked hard" | 学習目標志向、失敗後も粘り強さ維持、能力を伸ばせるものと捉える |

### 2.3 ⭐ ⚠ False Growth Mindset（Dweck本人の2015年警告）

Dweckが *Education Week* で**自身の理論の誤用を強く警告**：

#### 誤用1：戦略なしの「努力ほめ」は無意味
- 学習が進んでいないのに "good effort" と言うのは空っぽの励まし
- 結果が伴わない時は逆効果

#### 誤用2：マインドセットだけ持たせ戦略は教えない
- 「信じれば伸びる」は必要条件であって十分条件ではない
- 教師・保護者は学習者と一緒に**戦略を見つけ、成功をその戦略に帰属させる**長い道のりを歩む必要

#### 誤用3：「努力すれば誰でもできる」は誤り
- ディスレクシアは神経生物学的な特性
- 「もっと頑張れ」式の声かけは**二次障害（自尊感情低下・学習性無力感・不安・抑うつ）を強化する危険**
- 読み書きを「努力で克服する」というメッセージは Growth Mindset の**最悪の誤用**

### 2.4 Effective Effort（効果的な努力）

Dweck後年の修正：「努力 = 試行錯誤しながら**方略を変えていく**プロセス」

含まれるもの：
- **方法を変えること**（同じやり方で頑張らない）
- **助けを求めること**
- **ツールを使うこと**
- **戦略を変えること**

→ ディスレクシア指導では、**「努力ほめ」ではなく「戦略ほめ・工夫ほめ」へのシフト**が学術的にも推奨される。

### 2.5 ⭐ 声かけ実例

#### ✅ 良い例

**戦略ほめ**
- 「色付きの定規を試したのが効いたね」
- 「読み上げ機能を使う判断、上手だったね」
- 「行を指でなぞる工夫を続けてるの、いい方法だよ」

**プロセスほめ**
- 「最後まで諦めなかったね」
- 「わからない時に質問できたね」
- 「うまくいかないときに別の方法を試したね」

#### ❌ NGの例

**能力ほめ**
- 「君は頭がいいね」
- 「才能あるね」

**空虚な努力ほめ（戦略の言及なし）**
- 「頑張ったね」（結果が伴わない時は逆効果）

**誤用Growth Mindset**
- 「努力すれば読めるようになるよ」（ディスレクシアの神経学的特性を否定）
- 「やればできる」（戦略・方略の議論なし）

## 3. 自己効力感（Self-Efficacy）— Bandura（1977）

### 3.1 4つの源

| 源 | 内容 | 教育場面での実践 |
|---|---|---|
| **遂行行動の達成** | 実際に「できた」経験 | 最も強力。スモールステップで成功体験を積む |
| **代理経験** | 似た他者ができるのを見る | 同じ困難を持つ他者の体験談、ロールモデル |
| **言語的説得** | 信頼できる他者からの励まし | 「これだけ集中できるなら次もできると思う」 |
| **生理的・情動的状態** | 緊張・疲労の解釈 | 「ドキドキは挑戦の証拠」リフレーミング |

### 3.2 効力期待 vs 結果期待

これが**極めて重要な区別**：

| 期待 | 内容 |
|---|---|
| **効力期待** | 「自分は行動を遂行できる」 |
| **結果期待** | 「その行動をすれば結果が得られる」 |

ディスレクシア児は「努力しても成績が上がらない」経験から**結果期待が低下**：
- 「自分はできる」と思っても、「結果は出ない」と思っていれば動機は出ない
- → **効力期待だけ煽っても動機は回復しない**
- → **結果期待を高める**：「読みやすくなった事実」「読書時間が伸びた事実」を可視化

### 3.3 スモールステップで自己効力感を支える

「無理なく届く次の一歩」を設定：
- 「簡単なことから始める」のではなく、**子どもの現状を見極めて適切な難度に調整**
- 成功体験の**言語化**で「自分はできる」を内化させる

### 3.4 ⭐ 実例

- 成功体験：「昨日は3行、今日は4行読めたね。**1行増えたよ**」（具体的・比較可能）
- 代理体験：「同じディスレクシアの〇〇さんも、最初はこの方法から始めたんだよ」
- 言語的説得：「これだけ集中できるなら、次の段階も大丈夫だと思う」（**根拠を添える**）
- 生理的状態：「ドキドキするのは、新しいことに挑戦してる証拠だよ」

## 5. 学習意欲低下のメカニズム

### 5.1 学習性無力感（Seligman, 1967）
- 回避困難なストレス環境に長期間置かれた個体は、状況から逃れる努力すらしなくなる
- 「学習によって」獲得される無気力
- ディスレクシア児への含意：何年も読みで失敗を重ねた子は典型的に陥っている

#### 予防・回復
- **小さな随伴経験を積ませる**
- 「やってみたらうまくいった」を可視化
- 「行動と結果が連動する」体験を増やす

### 5.2 自己ハンディキャッピング（Berglas & Jones）
- 失敗の際に自尊心を守るため、あらかじめ自分に不利な条件を作る（「勉強してない」アピール）
- ディスレクシア児に多い「やる気がないフリ」「わざとふざける」行動の背景にあることが多い

### 5.3 失敗回避動機
- 成功への接近動機（成功したい）と並び立つ概念
- 失敗を恐れる気持ちが先行すると、挑戦そのものを避ける
- 過去の失敗経験が多い子ほど顕著

## 6. ⭐ 神戸大学・RIETI DP 22-J-037（2022）

「褒め方、叱り方が子どもの将来に与える影響―日本における実証研究」（神戸大学らの研究グループ）：

| 接し方 | 子の将来への影響 |
|---|---|
| **「頑張ったね」と過程ほめ** | **自己決定度・安心感が最も高い** |
| **「褒美をもらった」** | **自己決定度が最も低い**（アンダーマイニング） |
| **「次は頑張ろうね」と励まされた**（叱責時） | 「どうしてできないの」と叱られた人より自己決定度・安心感が高い |
| 「罰」「褒美」 | 長期的視点・倫理的行動が低下 |

→ アプリの声かけは**過程ほめ・戦略ほめを中核**に。

## 7. メタ認知・自己調整学習

### 7.1 メタ認知（Flavell, 1976）
- 「認知過程及びその関連事物に関する自分自身の知識」
- メタ認知的知識（何を知っているか）＋メタ認知的活動（モニタリング・コントロール）

### 7.2 自己調整学習（Zimmerman, 1989）

| 段階 | 内容 |
|---|---|
| **予見**（forethought） | 目標設定・計画 |
| **遂行**（performance） | モニタリング |
| **省察**（reflection） | 振り返り |

#### ディスレクシア指導への示唆
- 読み書きが困難でも、**「どの方略が自分に効くか」を意識する力（メタ認知）は伸ばせる**
- **ツール選択力・自己モニタリング力**を育てることが、長期的な自立につながる

#### メタ認知を促す3問テンプレ
- 「今日、何がうまくいった？」
- 「どの方法が合った？」
- 「次は何を変える？」

## 8. アンダーマイニング効果（Deci, 1971）

### 8.1 古典実験
- 大学生にパズル課題、Aグループには報酬、Bグループには報酬なし
- 自由時間中、報酬を受けた群はパズルへの興味が**低下**
- 内発的動機が外発的なものに置き換わり、報酬が無くなると行動も消える

### 8.2 教育場面での含意
- 「シール」「ご褒美」「バッジ」「スターポイント」の安易な導入は、興味があった活動への内発的動機を損なう
- ただし、もともと興味がない活動を始める「きっかけ」としては否定されない

### 8.3 使い分けの原則
- **興味がある活動** → 報酬を出さず、「過程ほめ」「戦略ほめ」で内発化を進める
- **興味がない活動** → 短期的な外的動機づけ可、徐々に内的価値を言語化し同一化的調整に移行

## 9. ⭐ 「強みベース」動機づけの学術根拠

Made by Dyslexia / NPO EDGE の「強みベース」アプローチは**3大理論で説明可能**：

| 理論 | 強みベースが機能する機序 |
|---|---|
| **SDT** | 有能感欲求が満たされる領域を確保。関係性も同時に支援 |
| **マインドセット** | 「自分は伸びる存在だ」を得意領域で確立し苦手領域に転移 |
| **自己効力感** | 別領域の成功体験が効力期待を一般化（spill-over effect） |
| **学習性無力感の反転** | 「行動と結果が連動する」経験を失敗の少ない領域で再構築 |

→ 単なる励まし哲学ではなく、**Bandura の言う「達成可能な領域での成功体験」を読み書き以外の領域で確保**することで、自己効力感の波及効果を狙う実践と再定式化できる。

## 10. ⭐ 声かけスクリプト集

### A. 学習開始時（自律性支援）

```
「今日は何から始めたい？選んでみよう」

「読む方法、自分で読む／読み上げ／誰かと一緒、どれが今日いい感じ？」

「うまくいかなくても大丈夫。やり方を変えればいいだけだから」

「『まだ』できないだけ。今日はその『まだ』を少し小さくする時間だよ」（"Yet" の活用）
```

### B. 取り組み中（戦略ほめ・有能感支援）

```
「指でなぞる方法、続けてるね。それが効いてるよ」

「わからない時に手を挙げられたの、上手だったね」

「読み上げを使う判断、自分でできたね」

「前は5分集中できなかったけど、今10分続けてるよ」（具体的・比較可能）
```

### C. うまく行かなかった時（学習性無力感の予防）

```
「今のやり方が合わなかっただけ。別の方法を試そう」（方略の問題に帰属）

「これは難しい問題だよ。読みづらいのは君のせいじゃなくて、
　フォントとか文字の並び方の問題でもあるんだ」（外的・特異的帰属）

「失敗じゃなくて、『この方法は効かない』っていう情報を一つ得たんだよ」
```

❌ NG例：「もっと頑張れば」「努力が足りない」「やればできる」（False Growth Mindset の典型）

### D. 成功体験の言語化（自己効力感の内化）

```
「今、何がうまくいったと思う？」（メタ認知促進）

「この工夫、君が自分で考えたんだよね。次にも使えそう？」

「3週間前の君はこれができなかった。今できてる。これは事実だよ」

「これができたなら、似たタイプの問題も同じ方法で行けそうだね」（一般化）
```

### E. 強みの言語化（ディスレクシア特有）

```
「君は人の話を聞いて全体像を掴むのがすごく早いね。それは大事な力だよ」

「文字で覚えるより、絵や動画で覚えるのが得意なんだね。それも立派な学び方」

「ひらめきが鋭いのは、君の脳の特性。読みは別のツールで補えばいい」
```

## 明日からできる3つのこと

1. **「努力すれば」式を使わずに、戦略・方略を具体的にほめる練習を1週間**
2. **「君は頭がいいね」「才能あるね」（能力ほめ）を使わず、「この工夫が効いたね」（戦略ほめ）に変える**
3. **「シール」「ご褒美」「バッジ」の安易な導入を避ける**
