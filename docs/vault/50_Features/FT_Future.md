# FT_Future — v2以降の将来機能

## v2 機能候補

### 1. OCR（カメラ／画像）
- 紙のプリント・教科書をカメラ撮影 → OCR → 読み上げ
- Google Cloud Vision / Tesseract.js
- Augmenta11y（ACIS 2021）の設計参考

### 2. RSVP（Rapid Serial Visual Presentation）
- 単語逐次提示モード
- 速度上限300 wpm（過剰速度は理解度悪化のため、PLOS One 2016 知見）
- デフォルトOFF・オプション

### 3. 教師・保護者ダッシュボード
- 生徒の使用設定を共有
- 合理的配慮申請の根拠資料を自動生成
- プロファイルの家庭⇔学校引き継ぎ

### 4. 講師向けLMS連携
- [[80_Teacher-Learning/Curriculum-Outline|Teacher Learning Curriculum]] をアプリ内に組み込み
- 進捗管理・修了証発行

### 5. 学校向けマルチユーザー版
- 教員アカウント／生徒アカウント
- クラスコード方式（個人情報不要）
- 教員が一括で配布・回収

### 6. 多言語対応
- 韓国語、中国語、東南アジア言語の追加
- 言語ごとのフォント・読み上げ・ガイドライン適用

### 7. アセスメント連携
- CARD等の標準化検査結果をインポート → 推奨設定を提案
- 大阪医科薬科大LDセンター等との共同研究

### 8. AI支援（オプション）
- 文章の難易度自動判定
- 「これはあなたには難しすぎるかもしれません」式の事前警告（任意）
- 「やさしい日本語」自動書き換え（[[FT_v1_1-Roadmap]]の発展）

### 9. アクセシビリティ強化
- スクリーンリーダー完全対応（VoiceOver、TalkBack、NVDA）
- キーボードのみで全操作可能
- 視線追跡入力対応
