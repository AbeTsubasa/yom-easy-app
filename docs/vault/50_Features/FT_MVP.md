# FT_MVP — MVP（v1.0）機能仕様

## アプリの一行コンセプト

> **ブラウザで即起動・登録不要・無料**で、テキストを **「自分に合う読みやすさ」** にカスタマイズし、**読み上げ・ルビ・行ハイライト**まで一括で使えるディスレクシア当事者のためのウェブツール。

## 設計原則（再掲）

1. 個人差を前提に「選べる」UIへ
2. デフォルトはWCAG AAAを上回る安全値で出荷
3. 「治す」フレーミング絶対回避（→ [[EM_Tone-Principles]]）
4. 「特別扱いに見えない」UI（→ [[F_Schools]]）
5. 講師も育つアプリ（→ [[80_Teacher-Learning/Curriculum-Outline]]）

## MVP機能セット

### A. 入力

- [ ] テキスト直接貼り付け
- [ ] .txt / .md ファイル読み込み（ドラッグ＆ドロップ）
- [ ] **🆕 カメラ撮影 → OCR**（Tesseract.js、日本語対応） — 詳細仕様は [[FT_OCR]]
  - スマホ：背面カメラ起動（`<input type="file" accept="image/*" capture="environment">`）
  - PC：ファイル選択／ドラッグ＆ドロップ
  - 抽出テキストを編集可能エリアに展開、視認性設定を自動適用

### B. 視認性コントロール（必須）

#### B-1. フォント
- [ ] フォント切替
  - 和文：UDデジタル教科書体（システム） / BIZ UDゴシック / Noto Sans JP / メイリオ
  - 欧文：Arial / Verdana / Open Sans / Helvetica / OpenDyslexic / Dyslexie
- [ ] フォントサイズ（14–32px、デフォルト18px）

#### B-2. 字間・行間
- [ ] letter-spacing（0–0.25em、デフォルト0.12em）
- [ ] line-height（1.2–2.0、デフォルト1.5）
- [ ] word-spacing（0–0.5em、デフォルト0.16em）
- [ ] paragraph-spacing（1–3em、デフォルト2em）

#### B-3. レイアウト
- [ ] max-width（30–50全角字 / 60–90半角字）
- [ ] text-align: left（固定推奨）

### C. 色（必須）

- [ ] 背景色プリセット：Cream `#FDF6E3` / Peach `#EDD1B0` / Orange `#EDDD6E` / Yellow `#F8FD89` / Light Grey `#D8D3D6` / Dark
- [ ] 文字色プリセット：Dark Grey `#222` / `#333` / Off-white (dark mode)
- [ ] 自由カラーピッカー
- [ ] ダークモード切替（純黒は使わず `#1a1a1a` ベース）

### D. 集中支援（必須）

#### D-1. TTS（読み上げ）
- [ ] Web Speech API による読み上げ
- [ ] 速度調整（80–150%）
- [ ] 一時停止・再開
- [ ] **単語同期ハイライト**（必須・bimodal効果のため）

#### D-2. Reading Ruler（行ハイライト）
- [ ] 3種類のスタイルから選択：Line Focus / Underline / Shade
- [ ] 色・高さ・透明度の調整

#### D-3. 蛍光マーカー
- [ ] 範囲選択 → 5色から選んでハイライト

### E. 日本語特化（必須）

- [ ] **ルビ自動付与**（kuromoji.js + kuroshiro）
- [ ] **学年別常用漢字フィルタ**（小1〜小6 / 中学 / 全部）
- [ ] 分かち書きトグル（ひらがな主体文で有効）

### F. オンボーディング・声かけ（必須・差別化軸）

- [ ] **初回起動時の3問テスト**（フォント／配色／行間を本人に選ばせる）
  - 「読みやすい方を選んでください」式、診断や評価とは無関係であることを明示
- [ ] **強みベースのメッセージング**を各所に組み込み（→ [[EM_In-App-Microcopy]]）
- [ ] Made by Dyslexiaの6スキル引用（任意・ON/OFF可能 → [[EM_Strength-Reminders]]）

### G. 出力

- [ ] 設定済みテキストの**印刷出力**（UDフォント・行間設定維持）
- [ ] 設定プロファイルの**URL共有**（合理的配慮申請の根拠資料にも使える）

### H. データ保存

- [ ] **localStorage** に個人設定を保存（アカウント不要）
- [ ] 設定の手動エクスポート（JSONダウンロード）
- [ ] **個人情報を一切収集しない**

## MVPで実装しないもの（v1.1以降）

→ [[FT_v1_1-Roadmap]] へ

- Bionic Reading風強調（オプション化）
- 文節改行（形態素解析応用）
- やさしい日本語化（LLM連携）
- PDFテキスト抽出
- OCRの高精度化（クラウドAPIオプション、手書き対応、縦書き対応）※基本OCRはMVP含む（[[FT_OCR]]）
- 教師ダッシュボード

## 受け入れ基準（Acceptance Criteria）

- [ ] WCAG 2.1 AAA 相当のテキスト視認性デフォルト
- [ ] **登録／ログインなしで全機能利用可能**
- [ ] スマホ・タブレット・PCいずれのブラウザでも動作
- [ ] **オフラインで基本機能（フォント・色・字間調整）が動く**
- [ ] **初回起動から30秒以内**にテキストを貼り付けて読み始められる
- [ ] 「治す」「直す」「克服」「練習」を1度も使わない
- [ ] ロゴ・アイコンに "Dyslexia" を含まない
- [ ] **カメラ撮影 → OCR → 読み上げのフロー**が日本語の印刷物で動作する
- [ ] OCR画像がサーバーに送信されていない（プライバシー検証）
