# F_Education-Boards — 教育委員会の発信

## 主要教育委員会の合理的配慮ガイド

| 自治体 | 文書 |
|---|---|
| **東京都教育委員会** | 『「読めた」「わかった」「できた」 読み書きアセスメント活用＆支援マニュアル』（小・中学校版）<br>「発達障害のある児童に対するICT機器等の導入ガイド」 |
| **千葉県教育委員会** | 「合理的配慮事例集〜通常学級に在籍する発達障害の可能性のある児童生徒〜」 |
| **山口県教育委員会** | 「学校における『合理的配慮』の提供」マニュアル（令和2年3月） |
| **大阪市教育センター** | 「ICT活用実践事例集（指導案等）」 |

## 京都市の発信

- **京都市LD等通級指導教室**：小・中学校の通常学級在籍のLD・ADHD・ASD児童生徒対象に自立活動を提供
- LD等発達障害保護者向けリーフレットを就学前〜小学校低学年保護者に配布
- 早期発見・早期支援を推進

## 自治体ガイドの典型的構成

1. LDの定義と典型的なつまずき
2. 早期スクリーニング・気づきのポイント
3. 通常学級／通級／特別支援学級それぞれの配慮例
4. 合理的配慮の申請プロセス
5. ICT活用の具体例
6. 保護者面談・本人との対話のヒント
7. 関係機関リスト（医療・福祉・専門相談）

## アプリ実装への示唆

- 設定プロファイル＋利用ログを「合理的配慮申請の根拠資料」として出力できると教員・保護者の負担軽減
- 自治体のガイドラインへのリンク集を [[TL_M5-Reasonable-Accommodation]] に集約
