# F_MEXT-NISE — 文部科学省・NISE

## 文部科学省

### 教育の情報化に関する手引（令和元年・追補版令和2年）
- 特別支援教育の章を拡充。発達障害カテゴリでICT活用の具体策を提示
- 読み書き困難児への代替手段：デジタル教科書の読み上げ・拡大、キーボード入力、写真によるノート代替、音声入力

### GIGAスクール構想と学習者用デジタル教科書
- 2024年度から本格導入。1人1台端末がインフラとして整備済み
- デジタル教科書搭載機能：**文字拡大／縮小、書き込み、背景色／文字色変更、行間調整、ルビ表示、機械音声読み上げ、ハイライト同期**
- 教科書バリアフリー法で「代替教材」として全課程で使用可
- **課題**：自治体・学校間の運用差が大きく、「特別扱い」と扱う学校もある

### 合理的配慮の3観点11項目（LDに対する例）
- ワープロ・PC・デジタルカメラの使用
- 口頭試問による評価
- 問題用紙の拡大、ひらがな解答容認
- 黒い下敷きの使用
- 高校入試・大学入試共通テストでも、別室受験・時間延長・読み上げ機能利用の実例あり

## NISE（国立特別支援教育総合研究所）

### 発達障害教育推進センター（cpedd.nise.go.jp）
- 「LDのある子どもの指導・支援」：つまずきの要因別指導法
- 「LDのある子どもの合理的配慮」：認知特性配慮で読み書きスキル向上の研究成果
- 教材・教具データベース、研修講義動画

### 代表事例（明治図書「教育zine」NISE連載）
特異的読み障害と診断された中学生Aさん：
- 視覚情報併用、問題用紙拡大、ひらがな解答容認、**黒い下敷きをテストでも認可**
- 通級でビジョントレーニング、タブレットOCR→読み上げ、音声入力作文
- SNS的文字コミュニケーションで抵抗感軽減

## アプリ実装への示唆

- **GIGAスクール環境（Chromebook主流）と親和性高いブラウザベース実装**
- デジタル教科書既存機能の「補完」として位置づけ（教科書外の任意テキストに対応）
- 合理的配慮申請の根拠資料を作成する機能（設定プロファイルの教員提示）
- 「特別扱いに見えない」UI（東京学芸大附属小金井小実践→[[F_Schools]]）
