# F_Universities — 大学発支援機関

## 大阪医科薬科大学LDセンター（奥村智人）

- 小児高次脳機能研究所附属
- オプトメトリスト・医学博士の**奥村智人**が視覚認知評価とビジョントレーニングを実施
- 就学前〜小学校低学年で「読みの方略」を身につけ視覚認知を高める基礎訓練
- **包括的領域別読み能力検査CARD**を2014年に開発
- ディスレクシアチェック「読めてる？」を監修
- UDデジタル教科書体の読み速度を一般教科書体と比較検証し**9%の改善**を報告（n=33）
- **思想**：「最終目標は定型発達児に追いつくことではなく**社会的自立**」

## 東京学芸大学

- 学校教育教員養成課程C類に「発達障害・学習障害系コース」設置
- 連合学校教育学研究科で発達支援・特別ニーズ教育・支援方法学の研究指導
- 附属小金井小学校の鈴木秀樹実践（→ [[F_Schools]]）が日本のインクルーシブ教育の先駆事例

## 慶應義塾大学（中野泰志研究室）

- **UDブラウザ**の開発元
- 弱視高校生・成人・教員ら200名超を対象とした評価でUDデジタル教科書体の可読性優位を実証

## 筑波大学・大阪教育大学（高橋登 等）

- 日本語ディスレクシアの基礎研究
- 高橋登：児童の読みの発達研究

## アプリ実装への示唆

- CARD など標準化されたアセスメントとの連携（v2以降）
- 「読めてる？」アプリ的なスクリーニング機能との差別化（こちらは**支援ツール**であり**診断ツールではない**）
- 大学研究室との連携で**査読論文の検証協力**が得られる可能性
