# V_Design-Dos-Donts — 当事者が望むデザイン／NGな表現

## 望むデザイン（DO）

| 項目 | 望ましい設計 |
|---|---|
| **メッセージ** | 強みベース。「直す／治す／克服」NG、「活かす／引き出す／自分らしく学ぶ」 |
| **見た目** | 中高生は「ディスレクシア用ツール」とバレない見た目。スタイリッシュ、汎用的 |
| **カスタマイズ自由度** | フォント・サイズ・行間・字間・背景色・文字色・読み上げ速度・ハイライト色を**個別**調整可能 |
| **読み上げ** | **同期ハイライトはほぼ必須**（DAISYで定着） |
| **マルチタスク負荷軽減** | 板書撮影→自動文字起こし、録音→テキスト化、ノート取得を代替 |
| **プライバシー** | 学校・友人に知られたくない／保護者にも内容を見られたくない。ローカル保存、ログ非可視化 |
| **価格** | オフライン動作・**無料**／低価格 |
| **テキスト整形** | 両端揃え禁止・左揃え、1行45-65文字、行間広め、視覚ノイズ排除 |
| **エラー時** | 責めず、代替方法を即示す |
| **成功時** | 派手な祝福ではなく、控えめに事実ベースで認める |

## やってはいけないこと（DON'T）

| NG | 理由 |
|---|---|
| **「治療」「直す」「克服」「正しく読めるように」表現** | ディスレクシアは脳の特性であり病気ではない。触るグリフも「治療法という言葉はミスマッチ」と明言 |
| **過度に医療的・診断的な表現** | 本人が「自分は壊れている」と感じる |
| **「がんばろう！」「乗り越えよう！」式の押し付け励まし** | 「もっとがんばれば読めるよ」が当事者を最も傷つけてきた歴史 |
| **子ども扱い／幼稚すぎるイラスト** | 中高生・大人当事者は「キャラクターが赤ちゃん向け」「色がパステルすぎる」「アニメ調」を嫌う |
| **ディスレクシアを前面に出す UI** | 「DYSLEXIA」ロゴ大、医療系の青十字風意匠は人前で開きたくない原因に |
| **画一的なフォント／配色を強制** | 「OpenDyslexic を使えばOK」ではない。エビデンスは限定的で、人によっては逆効果 |
| **大袈裟な啓発調コピー** | 「あなたは輝ける！」「強みを発揮しよう！」式は鬱陶しい。事実ベースの自然な言葉選びを |

## チェックリスト（実装前の検証用）

設計段階で以下を確認：

- [ ] ロゴ・アプリ名に "Dyslexia" "障害" などを含まない
- [ ] アイコンは中立的（医療系青十字、虹色、子ども向けキャラ NG）
- [ ] エラーメッセージが「あなたが悪い」感を出さない
- [ ] 励ましの言葉が「ここまでできたね」式の事実ベース
- [ ] 「治す」「直す」「練習」「訓練」「克服」を1度も使っていない
- [ ] フォント・色はユーザーが選べる（強制プリセットなし）
- [ ] スクリーンショット撮影時に「ディスレクシア専用」と分からない
- [ ] アプリを開いている画面を友人に見られても恥ずかしくない
