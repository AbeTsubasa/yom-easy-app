# 👩‍🏫 Teacher-Learning-MOC — 講師向け学習モジュール（M1〜M12）

> **このMOCの位置づけ**
> アプリを使う**講師（教員・保護者・支援者）**が、ディスレクシアのエビデンスと実践、そして**心理学に裏付けされた声かけ・対話**を段階的に学べるコンテンツ。

## 学習目標（要約）

修了後の到達目標：

### 基礎
- ディスレクシアを「治療すべき病気」ではなく「学び方の違い」として理解
- 個人差を前提に「複数の方法から本人が選ぶ」支援観
- 主要研究と公的ガイドライン数値の引用

### 応用 ★Phase 1-C
- 中田螺旋形モデルで保護者面談の6ステップを実践
- エビデンスベース介入を受験生に提案
- **False Growth Mindsetの罠**を回避し戦略ほめへ
- SAMHSA 6原則で「再受傷させない」教室を設計
- TALK の原則で緊急対応
- MI・OARS・Rogers・GROW・SFA を場面別に使い分け
- Wait-Time 3秒以上の実践
- 共感疲労・代理受傷・バーンアウトの自己モニタリング

## 推奨ルート（用途別）

- **塾長・予備校講師**：M2 → M8 → M9 → M11 → M10 → M12 → 残り
- **学校教員**：M1 → M2 → M3 → M4 → M5 → M6 → M7〜M12
- **緊急時最優先**：M2 → M7 → M10 → M11 → M12 → 残り
- **エビデンス重視**：M1 → M3 → M9 → M4 → M8 → 残り
