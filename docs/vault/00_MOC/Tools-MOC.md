## 横断キー知見：参入余地

1. 「ブラウザで即起動・申請不要・無料」で日本語ルビ＋読み上げ＋UDフォント＋行ハイライトを統合したツールが**事実上存在しない**
2. 公的ツール（DAISY/AccessReading/UDブラウザ）はいずれも**申請制**で日常使い不可
3. 海外ツールは**日本語ルビ自動付与に未対応**
4. 既存ふりがな拡張は**単機能**で統合性なし
5. 商用学習アプリは「教材内」読み上げのみで**外部テキスト不可**

→ 推奨ポジショニング：**「ブラウザで即起動・登録不要・無料・日本語ルビ／読み上げ／UDフォント／行ハイライト統合」**
