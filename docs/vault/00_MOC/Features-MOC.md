## MVP差別化軸（5つ）

1. **日本語ルビ＋読み上げ＋UDフォントの統合** — 既存ツールにない空白を埋める
2. **ブラウザで即起動・登録不要・無料** — 日常使いを阻害しない
3. **徹底的なカスタマイズ自由度＋プリセット** — 個人差を前提
4. **「特別扱いに見えない」UI／強みベースのメッセージング** — 当事者の声を反映
5. **学校現場で使える前提条件をクリア** — 印刷・共有・プライバシー
