# Yom-easy ベータ版 フィードバックフォーム雛形

Google Forms にそのまま貼って使える形にしてあります。質問数は 5 つまでに絞っています（読む側・書く側の負荷を最小化）。

---

## フォームの設定（Google Forms 側で）

- **タイトル**：Yom-easy を試していただきありがとうございます
- **匿名回答**：ON（メールアドレス収集 OFF）
- **回答制限**：1 人 1 回（OFF が無難。複数回答えてもらってよい）
- **回答先のシート**：塾長の個人 Google アカウントの非公開シート

---

## フォーム冒頭の説明文（テスター向け）

```
試してくださってありがとうございます。

このフォームは、Yom-easy をもっと使いやすくするための声を集めるためのものです。
答えなくて大丈夫な項目もあります。書きたいことだけ、書きたい分だけ書いてください。

「合わなかった」「使いづらかった」は、ありがたい声です。
そのまま書いていただいて大丈夫です。

メールアドレスやお名前は集めません。すべて匿名で受け取ります。
```

---

## 質問（5 問）

### Q1. 試してみて、目（や頭）はどうでしたか？

**形式**：単一選択（5 段階）

- とても楽になった
- 少し楽になった
- 変わらない
- 少し疲れた
- とても疲れた

※「答えない」も選べる選択肢として「分からない／答えたくない」を追加してもよい

---

### Q2. うれしかった機能、合っていた設定はありますか？（いくつでも）

**形式**：複数選択

- フォントを変えられたこと（UD 教科書体・Open Sans など）
- 文字の大きさ・字間・行間・行幅を変えられたこと
- 背景や文字の色を変えられたこと
- 行ハイライト（1 行ごと／全行）
- 段落フォーカス（読んでいる段落以外を薄める）
- ふりがな
- 分かち書き
- 文節改行
- 読み上げ
- 設定の共有 URL
- カメラで撮って文字を読み取る機能
- どれも特に合わなかった
- 分からない

---

### Q3. 困った瞬間、引っかかった瞬間はありましたか？

**形式**：自由記述（複数行、必須なし）

```
例：
・どこから触ればいいか迷った
・◯◯ の機能を ON にしたら、見にくくなった
・ボタンの場所が分かりにくかった
・スマホでうまく表示されなかった
・◯◯ の言葉が分かりにくかった
など、思いついたことを、どんなことでも教えてください。
```

---

### Q4. 「こうだったらいいのに」と思うことはありますか？

**形式**：自由記述（複数行、必須なし）

```
こんな機能があったら、もっと使いたい、もっと楽になるかも、
というアイデアがあったら教えてください。
（実現できるかは別ですが、声はすべて見させていただきます）
```

---

### Q5. 自由欄

**形式**：自由記述（複数行、必須なし）

```
ここまでに書ききれなかったこと、感想、励まし、苦情、何でも。
書きたくなければ空欄で大丈夫です。
```

## フォーム末尾のメッセージ

```
ありがとうございました。

ここで集めた声は、私（塾長）が一つひとつ拝見し、次の改良に活かします。
すぐに改良できるとは限りませんが、必ず読みます。

体験のあとに、もし辛い気持ちが残ったら、
無理せず画面を閉じて、お茶でも飲んでください。

塾長 阿部翼
```

---

## 運用メモ（塾長向け）

- **集計を急がない**。1 週間に 1 度まとめて見れば十分です
- **「合わなかった」回答に落ち込まない**。これがいちばん貴重な情報です
- **個別に返事をしなくてもよい**。匿名なので返せないことも多いはず。返さないことが失礼にならない設計
- **回答数が少なくても気にしない**。ベータ期間は「使ってもらえただけで成功」と考えてください

強く促さず、「もしよかったら、こちらに」のトーンで添えてください。
